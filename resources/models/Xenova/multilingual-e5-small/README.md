# Xenova/multilingual-e5-small

Default Smart Assist embedding model for v236.

Put the downloaded model files for `Xenova/multilingual-e5-small` in this directory, or use the Smart Assist admin screen to download the model to an arbitrary folder.

Required files:

```txt
config.json
tokenizer.json
tokenizer_config.json
special_tokens_map.json
onnx/model_quantized.onnx
```
