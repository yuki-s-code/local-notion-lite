# V383 添付アップロード軽量化

## 目的

BlockNote のドラッグ&ドロップ添付はブラウザの `File` から base64 を作り、localhost API へ JSON として送ります。大きな PDF では Renderer と Node の両方で一時的に大きなメモリを使うため、本文へのドラッグ追加を 15MiB までに制限します。

## 運用

- 15MiB 以下: 本文へドラッグ&ドロップ可能。
- 15MiB 超: 「添付を追加」ボタンからOSのファイル選択を使用。Electron のローカルパスをサーバーへ渡すため、base64 化しない。
- サーバー側でも同じ上限を検証し、古いRendererや直接API呼出しを防御する。

## 保存

base64 経由の本文添付ファイルも、一時ファイルへ書いてから完成ファイルへ移動する。途中失敗時は `.tmp` を掃除し、添付Indexへ登録しない。
