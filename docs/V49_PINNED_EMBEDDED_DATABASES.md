# v49 Pinned Embedded Databases

長い本文を書き進めると、本文内に埋め込んだデータベースが画面外へ流れて見失われる問題を改善しました。

## 追加内容

- 本文編集エリアの右側に、埋め込みデータベース用の固定レールを追加
- 本文を下へ書き進めても、データベースを常に確認可能
- 複数のDBを埋め込んだ場合はタブで切替
- 狭い画面では本文上部に折りたたみ可能な固定カードとして表示
- 表示専用の軽量DBビューとして、行操作UIや詳細ペインは非表示

## 方針

本文エディタそのものはBlockNoteのルールに従い、DOMを直接改変しません。  
DBの追従表示はReact側のレイアウトで実現しています。
