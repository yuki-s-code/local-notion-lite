# v54 Advanced Database Table

## 追加内容

- TanStack Table の列幅変更を有効化
- 列表示/非表示設定を追加
- 列幅と列表示設定を localStorage に保存
- 行選択チェックボックスを追加
- 選択行のCSV保存、複製、削除を追加
- 分析グラフを「概要 / 数値 / 分類 / 日付」で切替可能に改善
- テーブル設定パネルを追加

## 方針

データベース本体のJSON互換性は維持し、表示設定は端末ごとのlocalStorageへ保存します。
共有フォルダに正本データを置く現在の設計を壊さず、各PCで使いやすい表示設定を保持できます。
