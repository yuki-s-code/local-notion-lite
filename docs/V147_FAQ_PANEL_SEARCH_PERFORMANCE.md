# v147 FAQ Panel / Search Performance

## 目的
- Local AI Chat右側FAQパネルのはみ出しを修正
- FAQが大量になっても画面描画が重くなりにくいよう、FAQ一覧の表示件数を段階表示に変更
- FAQ検索の入力を遅延反映し、入力中の再スコアリング負荷を軽減

## 変更点
- 右側パネルに `min-width:0` / `max-width:100%` / `overflow-x:hidden` を追加
- FAQ統計カードを2列基準にして、狭い右サイドバーでもはみ出しにくく調整
- FAQ検索欄とステータス選択を `minmax` レイアウトに変更
- FAQ一覧は初期20件表示、以降「さらに表示」で20件ずつ追加
- FAQ検索語は180msの遅延反映
- FAQリストに `contain` を付与して描画影響を抑制

## 運用目安
- 数百件: 問題なく運用しやすい
- 数千件: 検索 + 20件段階表示で運用推奨
- 1万件以上: 次段階でFAQ専用SQLite/FTSインデックス化推奨
