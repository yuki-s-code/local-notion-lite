# V356 AI Source Grounding Control

右下AIチャットの参照元制御を強化しました。

## 変更点

- 「使う」にした参照元を、次の質問でも確実にAI回答の文脈へ渡すように変更
- 「固定した参照元だけで回答」チェックを追加
- 固定参照元は検索結果から外れても回答文脈に残るように変更
- 回答ごとに根拠状態を表示
  - 根拠: 高 / 中 / 低 / なし
  - 使用した参照元数
  - 固定参照元数
  - 除外参照元数
  - 固定のみモード
- サーバー側でも `sourceMode: pinned_only` に対応

## 目的

業務用途では、AIが自動検索した候補を勝手に混ぜるより、ユーザーが指定した根拠だけで回答したい場面があります。
V356では「この資料だけを根拠にして」「このFAQは除外して」という使い方がしやすくなります。

## 使い方

1. 参照候補カードで「使う」を押す
2. 必要なら「固定した参照元だけで回答」をON
3. 次の質問を送る

ONの場合、固定した参照元だけを根拠に回答します。
OFFの場合、固定した参照元を強く優先しつつ、通常検索候補も補助的に使います。
