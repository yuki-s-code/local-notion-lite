# V143 Local Generative Assist UX

## 目的
Local Generative Assistをユーザーに分かりやすいチャット中心画面へ整理し、現在登録されているFAQの状況を一目で確認できるようにした。

## 追加・改善

- 画面上部に「質問する → 根拠を見る → FAQに育てる」の3ステップガイドを追加。
- 右サイドバーをFAQライブラリ中心に再構成。
- FAQ件数、承認済み、確認済み、下書き、PDF由来数をカード表示。
- FAQ検索、ステータス絞り込み、カテゴリチップを追加。
- FAQ一覧からそのまま質問・編集画面へ移動できる導線を追加。
- 回答品質、参照元、自動FAQ候補を整理して表示。
- FAQ管理画面にもFAQサマリーを追加。

## 運用意図

日常利用ではチャットとFAQライブラリだけで完結し、詳細検索・形態素風トークン・診断は必要なときだけ開く構成にする。
