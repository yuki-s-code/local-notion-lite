# v208 Chat Answer Safeguard

## 目的

v207で最終ランキングの再ソートを入れた後、環境によってチャット回答が空になるケースに対応しました。

## 修正内容

- `/smart-assist/chat/ask` で例外が起きても、UIが空にならないように回答JSONを返す
- `answer` が空のレスポンスを検知して、安全な低信頼回答へ変換
- Guarded Ranking が候補を空にした場合、pre-guard の候補へフォールバック
- confidence 計算で NaN が混じっても落ちないように補正
- mode を `transformer-semantic-hybrid-v208` に更新

## 確認する質問例

- クラブの費用を教えて
- 学童の費用はどのように確認するか
- 有給はいつから取得できますか
- 申請した後に取り消したい

## 補足

今回の修正は、精度ロジックそのものを大きく変えるものではなく、回答が空になる障害を防ぐ安全策です。
