# V427 タグ別ページ一覧（専用ビュー）

## 目的
タグに一致するページを、単純な検索結果ではなく、一覧として閲覧・整理できる専用ビューにします。

## 実装内容
- タグ管理画面の関連ページ領域を「タグ別ページ一覧」に刷新
- 複数タグの AND 条件を維持
- タイトル・タグによる追加絞り込み
- タググループ（業務分野、年度、対象者、状態、その他）による絞り込み
- 更新が新しい順、作成が新しい順、タイトル順のソート
- 一覧表示とカード表示の切替
- タグ色・グループ設定をチップ表示へ反映
- ページをクリックして既存のページ表示へ移動

## 安全性
このビューは読み取り専用です。本文、ページタグ、コメント、履歴、共有辞書は変更しません。

## 補足
表示条件は現在開いているタグ管理画面内の状態として保持します。ページ数が多い場合も、既に読み込まれているページメタデータだけを対象にするため、追加の全件サーバー読込は発生しません。
