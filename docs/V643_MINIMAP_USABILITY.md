# V643 — ミニマップの移動と透明表示

## 変更内容

- ミニマップ上部の持ち手をドラッグして、キャンバス内の任意位置へ移動できるようにした。
- ミニマップ本文は従来どおり、クリックすると表示位置を移動する。ドラッグ操作と地図クリックの役割を分離した。
- ミニマップは通常時 `opacity: .78`、ホバー・キーボードフォーカス時は `opacity: .96` にし、地図を隠しにくくした。
- 位置は端末内 `localStorage` のみへ保存する。共有フォルダ・ページデータ・同期状態には影響しない。
- 持ち手のダブルクリックで、右下の既定位置へ戻せる。

## 3Dについて

日常利用は2Dを維持する。3DはWebGL/GPUと奥行き操作の複雑さが増し、共有PCやIntel内蔵GPUを含む環境での操作安定性・可読性を下げやすいため、全体マップの試験的表示以外では採用しない。
