# v157 Question Typography Balance

## Purpose
Recommended questions and pinned questions were visually stronger than the chat body. This version reduces font size, padding, and visual weight so those controls remain useful without dominating the Local AI Chat screen.

## Changes
- Reduced typography size for recommended question chips.
- Reduced pinned question row font size and button height.
- Tightened spacing in the recommended question strip.
- Tightened spacing in the pinned question card.
- Added responsive limits for narrow screens.

## Scope
UI/CSS only. No FAQ, chat, search, or answer logic was changed.
