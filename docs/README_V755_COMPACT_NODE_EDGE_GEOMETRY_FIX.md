# V755 Compact Node Edge Geometry Fix

## 修正内容

- ズームアウト時にページ・データベース・PDFなどがタイトルのみの低詳細表示になる場合、接続線も実際の表示高さ（54px）を使って端点を計算するように修正。
- 保存されている展開時サイズは変更せず、表示専用ノード寸法を生成する方式にしたため、ズームを戻すと元のカードサイズと接続位置へ戻る。
- 選択中のカードは詳細表示されるため、選択状態に応じて接続線の端点も即時更新。
- Viewport仮想化、論理グループ枠、接続線レイヤーが同じ表示用ノード寸法を参照するよう統一。
- 選択IDのSet生成を`useMemo`で共通化し、表示計算内の重複生成を削減。

## 原因

CSSでは低詳細表示時にカードの実表示高さを54pxへ縮小していましたが、接続線は永続化された展開時の`node.h`を参照していました。そのため、タイトルだけのカード枠ではなく、見えない展開時領域の上下左右から線が出ていました。
