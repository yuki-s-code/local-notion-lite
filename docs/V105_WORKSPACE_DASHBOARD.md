# v105 Workspace Dashboard / Productivity Hub

## Added

- Home dashboard for daily entry point
- Recent pages / journals / inbox / tasks / attachments summary
- Global attachment manager
- Link management screen for broken local links and orphan candidates
- Workspace admin screen for sync, SQLite, trash and conflict status
- Page outline side utility for long BlockNote pages

## Design notes

This version intentionally keeps the BlockNote editor stable. New features are implemented around the editor rather than by mutating the editor instance during typing.
