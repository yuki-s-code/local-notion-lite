# v422 Smart Assist タグ優先検索

## 追加内容
- ページのプロパティタグをWorkspace Semantic Indexの `tags` と `keywords` に登録します。
- Smart Assistは質問文と共有タグ別名辞書から関連タグを推定し、最大8件を検索リクエストへ渡します。
- Semantic検索後の再ランキングで、根拠チャンクのタグと一致する候補を最大24点加点します。
- 回答の下に関連タグを表示し、質問一致・根拠一致と使用ページ数を確認できます。
- タグチップを押すと、次の質問欄へタグを入れられます。自動送信はしません。

## 運用
この更新後、Smart Assist管理画面のSemantic Indexで「差分更新」を実行してください。既存インデックスには過去のページタグが未登録のためです。

## 安全性
- タグは検索・表示専用で、質問や回答によってページタグを自動変更しません。
- タグ候補は既存ページタグ、FAQタグ、共有別名辞書にある正式タグからのみ決定します。
