# v22 Sidebar / BlockNote Polish

## 目的

v21までのNotion風直接編集を前提に、サイドバー操作と本文編集体験をさらにNotion寄りにした版です。

## 追加・変更

- サイドバーでページをお気に入り登録できるようにしました。
- お気に入りページをサイドバー上部に表示します。
- ページツリー内で子ページ作成ボタンを強化しました。
- ページをドラッグして別ページへドロップすると、そのページの子ページへ移動できます。
- ページをルートへ戻すためのドロップゾーンを追加しました。
- 同階層内の並び替え用に上/下ボタンを追加しました。
- `@` リンク挿入時に行が一段下へずれにくいよう、BlockNoteのインライン挿入を優先するようにしました。
- 旧来の別管理添付ファイルUIを画面から削除しました。添付はBlockNote内部のファイル/画像ブロックへ寄せていく方針です。
- BlockNote編集画面を白背景に固定しました。
- サイドバーのステータス/優先度バッジ常時表示を抑え、ページ本文側のプロパティ折りたたみを基本にしました。

## 使い方

### お気に入り

ページツリーの星ボタン、またはページ上部の「お気に入り」ボタンから切り替えます。

### ページ移動

ページ行をドラッグし、移動先ページの上にドロップすると、そのページの子ページになります。
ルートへ戻したい場合は、ページツリー上部の「ここにドロップしてルートへ移動」にドロップします。

### 並び替え

ページ行にマウスを乗せると、上/下ボタンが表示されます。同じ親を持つページ同士で並び替えできます。

## 注意

共有フォルダ運用の安全性を優先して、リアルタイム共同編集ではなくページ単位ロック方式を維持しています。
