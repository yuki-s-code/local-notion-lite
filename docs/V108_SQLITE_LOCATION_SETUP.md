# v108 SQLite保存先設定

## 追加内容

- 初回起動時にSQLiteキャッシュ保存先を選択可能にしました。
- 自動選択も選べます。
- 設定画面からSQLite保存先を変更できます。
- 設定画面から自動選択へ戻せます。

## 設計

共有フォルダのJSON/Markdown/attachmentsが正本です。SQLiteは各PC用の検索・表示キャッシュです。

そのため、SQLite保存先を変更した場合はアプリ再起動後に新しい場所の `local.sqlite` を使います。必要に応じて共有フォルダから再同期してください。

## 保存先

任意フォルダを選択すると、その中に以下を作成します。

```txt
選択フォルダ/local.sqlite
```

`.sqlite` または `.db` ファイルパスが設定されている場合は、そのファイルを直接使います。
