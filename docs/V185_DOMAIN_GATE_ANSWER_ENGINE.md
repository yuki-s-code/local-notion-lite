# V185 Domain Gate Answer Engine

曖昧・汎用FAQチャットの回答精度を上げるため、回答生成前に「質問意図に合わないFAQ」を除外するドメインゲートを追加しました。

## 主な改善

- 「申請期限が過ぎた」などの期限超過・申請遅れ意図を追加
- 「勤務時間を変更したい」などの変更意図を追加
- 子の看護、扶養、通勤、就労要件など別分野のFAQが回答本文へ混ざる問題を抑制
- 回答生成前に候補FAQを絞り込み、回答表示直前にも不要行を削除
- サーバー検索側でも期限超過意図のスコア補正を追加

## 期待する挙動

「申請期限が過ぎてしまった」には、期限後の受付可否、再申請、次回受付、担当窓口への確認を中心に回答します。

「勤務時間を変更したい場合は？」には、変更届、事前承認、必要書類、反映時期を中心に回答します。
