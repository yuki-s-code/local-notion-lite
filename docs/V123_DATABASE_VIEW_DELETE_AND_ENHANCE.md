# v123 Database View Delete and Enhance

## 修正内容

- Board / Calendar / Gallery / Timeline を押すたびに同じ種類のビューが増える挙動を修正。
  - 既に同じ種類のビューがある場合は、そのビューへ切り替えます。
  - 存在しない種類だけ新規作成します。
- ビュータブ上に削除ボタン（×）を追加。
  - 最後の1ビューは削除できません。
  - 削除時は確認ダイアログを表示します。
  - ビュー削除では行データは削除されません。
- View設定内の削除ボタンも確認付き削除に変更。

## データベース機能強化

- 「今日の行」ボタンを追加。
  - Dateプロパティには今日の日付を自動入力します。
  - Status系Selectには先頭候補を自動入力します。
- プロパティカードで列タイプを変更可能にしました。
- Select / Multi Select の選択肢編集を追加しました。
- Multi Select セルをタグ風チェックUIで編集できるようにしました。
- View設定に「条件クリア」を追加し、Filter / Sort をまとめて解除できるようにしました。

## 注意

v123では「ビュー種別ボタン」は追加ボタンではなく、スマート切替ボタンとして扱います。複数の同種ビューを作りたい場合は、View設定の複製ボタンを使います。
