# V324 Notion-like AI Search Orb

## Summary

The Workspace AI Search floating entry point was changed from a text-heavy pill button to a compact Notion-like AI orb.

## Changes

- Replaced the visible `✨ AI検索` pill with a custom inline SVG icon button.
- Added an accessible `aria-label` and kept the native title tooltip.
- Added a small lower label for discoverability on desktop.
- Added a hover/focus tooltip on the left side.
- Added glassy/minimal styling, hover glow, and responsive mobile behavior.

## Intent

The button should feel like a lightweight assistant entry point rather than a large CTA. This keeps the workspace cleaner while preserving discoverability.
