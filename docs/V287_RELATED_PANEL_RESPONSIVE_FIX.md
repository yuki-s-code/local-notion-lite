# v287 Related Panel Responsive Fix

## 目的
v285/v286で追加したruri-v3関連情報パネルが、画面幅・リンクプレビュー表示・DB行プレビューのリサイズ幅によって非表示になる問題を修正する。

## 原因
既存のv105レスポンシブルールで、`max-width: 1100px` 以下の場合に `.page-right-utility-v105` が `display: none` になっていた。リンクプレビューや狭いウィンドウではこの条件に入り、関連情報パネルごと非表示になっていた。

## 修正方針
関連情報パネルは実務上のナビゲーション機能なので、狭い画面でも消さず、右レールから本文上部のコンパクトカードへフォールバックさせる。

## 変更点
- `.page-right-utility-v105` を狭い幅でも表示するように上書き
- リンクプレビュー表示時も関連情報パネルを本文上部へ移動
- 720px以下では説明文・一部メタ情報を省略してコンパクト化
- DB行プレビューのリサイズ時も関連カードを非表示にしない

## 今後
関連パネルの情報量が増えた場合は、CSSだけでなくコンポーネント側に `displayMode="rail|card|compact"` を追加して、UI密度を明示的に制御する。
