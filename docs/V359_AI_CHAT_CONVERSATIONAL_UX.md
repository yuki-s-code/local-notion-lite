# V359 AI Chat Conversational UX

右下AIをよりチャットボットらしくするため、会話UXを強化しました。

## 追加内容

- 口調プリセット
  - スマート
  - やさしく
  - 業務メモ
  - 保護者向け
  - 職員向け
- 直近会話の文脈をAI回答に渡す
- 曖昧な質問では無理に断定せず聞き返す
- 回答後に「続けてできます」を表示
- 根拠・参照候補を折りたたみ表示に変更
- 根拠0件時の文面をチャット向けに改善

## 目的

行政文書のような固定見出し回答ではなく、自然な会話、聞き返し、次の操作提案ができるAIアシスタントに近づけること。
