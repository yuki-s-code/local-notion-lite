# V180 Generic FAQ Answer Engine

## 目的

芦屋市学童FAQだけに寄らず、休暇・給与・手続き・IT・マニュアル・PDF由来FAQなど、どのジャンルでも使える汎用FAQ回答エンジンへ拡張しました。

## 主な変更

- 条件・対象、手続き、必要書類、料金、期限、変更、困りごと、概要の汎用意図プロファイルを追加
- FAQ回答文から、質問意図に合う文を抽出して回答本文だけを生成
- 関連FAQ・関連ページは本文に混ぜず、クリック可能な source ボタンとして残す
- 学童専用語だけに依存しないよう、汎用同義語と汎用スコア補正を追加

## 回答方針

1. まずFAQ検索で候補を取得
2. 質問文と候補FAQから汎用意図を判定
3. 回答本文から最も質問意図に近い文を抽出
4. 回答本文はシンプルに表示
5. 関連情報はクリック可能なボタンとして表示

## 想定質問

- これは対象になる？
- どうしたらいい？
- 必要なものは？
- いつまで？
- いくらかかる？
- 変更したい
- 困ったときは？
- 違いを教えて
