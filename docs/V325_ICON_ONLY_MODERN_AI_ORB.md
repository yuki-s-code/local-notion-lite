# v325 Icon-only Modern AI Search Orb

## 目的
右下のWorkspace AI Search入口を、文字付きボタンから、Notion風・モダン・ミニマルなアイコン専用ボタンへ変更した。

## 変更点
- 右下AI検索ボタンから表示文字を削除
- `notion-ai-orb-v325` クラスを追加
- インラインSVGを新しいスパーク/オーブ形状に刷新
- ホバー時の文字ツールチップも廃止し、完全なアイコン専用UIに変更
- ガラス質感、立体的な影、内側ハイライト、控えめなグローを追加
- ダークモード用の見た目を追加
- モバイル幅では50pxのコンパクトサイズに調整
- `aria-label` と `title` は維持し、アクセシビリティと説明性は保持

## 方針
目立ちすぎず、画面右下に自然に置けるAI入口にする。本文やDB操作の邪魔をせず、必要な時だけ視線に入るデザインを優先した。
