# v18 Notion-like UX

## 追加内容

- Ctrl/⌘ + K のコマンドパレット
- ページ・データベースの横断検索
- パンくずリスト
- Notion風カバー表示
- ページアイコン編集
- 固定ツールバー
- @入力中候補の改善
  - `@` だけでも候補表示
  - `@タイトル` で候補絞り込み
- `/database` / `/db` 入力中候補の改善
  - `/database` だけでもDB候補表示
- ページリンクパネルとDB埋め込みカードの見た目改善

## 保存形式

ページ本文は引き続き BlockNote 形式です。

```json
{
  "version": 1,
  "kind": "blocknote",
  "blocks": []
}
```

ページアイコンは `meta.json` の `icon` に保存されます。

## 操作

- `Ctrl + K` または `⌘ + K`：コマンドパレット
- 本文で `@`：ページ候補
- 本文で `@タイトル`：ページ候補絞り込み
- 本文で `/database`：データベース候補
- 本文で `/db タイトル`：データベース候補絞り込み

## 注意

v18でも、@リンクやDB埋め込みは安全性重視のマーカー方式です。

- ページリンク: `@[[タイトル|page_id]]`
- DB埋め込み: `{{database:db_id}}`

次の段階では、BlockNoteのカスタムスラッシュメニューやインラインメンション化に進めます。
