# v735 BlockSuite風ホワイトボード強化

## 概要
v734のFreeform Canvasを、BlockSuite/AFFiNE風の「ブロックを置いて考えるホワイトボード」に近づけた。
既存のナレッジマップとは重複させず、手動整理・発想・図解・関係づけに特化している。

## 主な変更
- フローティングツールバーを追加
  - 選択
  - 移動
  - 付箋
  - テキスト
  - 図形
  - 接続
- ダブルクリック配置を追加
  - 付箋/テキスト/図形ツール時、キャンバス空白をダブルクリックして追加
- 図形ブロックを追加
  - 角丸
  - 四角
  - 楕円
  - ひし形
- 接続モードを追加
  - 接続元カードをクリック
  - 接続先カードをクリック
  - 2カードを線で接続
- テキストブロックの見た目をカードではなく軽いテキスト風に変更
- 図形/フレーム/付箋の見た目をBlockSuite風に調整
- 既存localStorageを読むフォールバックを残しつつ、新しい保存キーへ移行

## 効率面
- 新しいライブラリ追加なし
- サーバーAPI追加なし
- 保存は従来どおりlocalStorage + debounce
- 開いていない時の追加負荷なし
- ナレッジマップやDBには影響させない
