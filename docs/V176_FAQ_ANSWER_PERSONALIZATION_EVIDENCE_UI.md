# v176 FAQ Answer Personalization + Evidence UI

## 目的
v175で改善したFAQ検索ランキングに加えて、チャット回答のユーザー体験を改善する。

## 変更内容

### 1. FAQ回答のパーソナライズ
質問文の意図に合わせて、FAQ本文をそのまま出すだけでなく、先頭に直接回答を表示する。

例：
- 「週何日働いていたら？」→ 週3日以上を先に表示
- 「16時前に終わる場合は？」→ 午後4時までに帰宅困難な場合の扱いを先に表示
- 「必要書類は？」→ 就労証明書・シフト表・自営業資料を整理して表示
- 「料金は？」→ 育成料・減免・兄弟減額を整理して表示

### 2. 要点表示
FAQ本文から、ユーザーが見たい要点を箇条書きで表示する。

### 3. 根拠FAQの明示
回答の下に、根拠FAQ、分類、状態、スコア、一致理由、一致語、参照元を表示する。

### 4. 次に確認すると良い質問
回答後に、関連する次の質問候補を表示する。

### 5. 検索ランキングの追加補正
就労、料金、申請、必要書類、利用時間、学年、退会、変更、空き状況などの意図を追加判定し、概要FAQが上に来すぎる問題を抑制する。

## 方針
外部AIは使わず、ローカルFAQ検索結果だけで回答する。
生成AI風の自然な体験を出すが、根拠FAQを必ず表示することで業務利用の安全性を維持する。
