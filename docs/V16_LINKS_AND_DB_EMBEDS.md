# v16: @ページリンク / データベース埋め込み

## 追加内容

- BlockNoteエディタ上部に `@ ページリンク` 検索を追加
- ページを選ぶと `@[[ページ名|page_id]]` 形式のリンクブロックを挿入
- BlockNoteエディタ上部に `/ データベース` 検索を追加
- データベースを選ぶと `{{database:db_id}}` 形式の埋め込みブロックを挿入
- 本文内のページリンクを自動抽出し、クリック可能なリンク一覧を表示
- 本文内のデータベース埋め込みを自動抽出し、読み取り専用テーブルとしてページ下部に表示

## 現時点の仕様

BlockNote本体のインラインMentionやカスタムブロックへ完全統合する前段階として、まず共有フォルダ保存と後方互換を壊さないマーカー方式で実装しています。

- ページリンク: `@[[タイトル|page_id]]`
- データベース埋め込み: `{{database:db_id}}`

この方式はMarkdownにも残るため、検索・履歴・差分・競合退避と相性が良いです。
