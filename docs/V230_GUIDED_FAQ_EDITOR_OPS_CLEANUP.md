# v230 Guided FAQ Editor / Operations Cleanup

## 目的
FAQ管理画面の追加・編集操作を、入力者が迷わない専用モーダルに再構成しました。あわせて、FAQ管理と運用に重複していた操作を整理しました。

## 変更内容

- FAQ Editorを4ステップ型に再構成
  - 質問と回答
  - 検索ヒント
  - 誤回答ガード
  - 根拠
- 入力欄ごとに「何を入れるか」が分かる説明と例を追加
- likelyQuestions / paraphrases / testQuestions / negativeTerms の意味が分かるように整理
- 出典タイトル、ページ、章・節、URLを入力できるように整理
- FAQ管理にある評価セット自動生成・正答率測定・検索再生成を中心機能として維持
- 運用タブは保守操作だけに整理
  - 検索・意味ベクトル再生成
  - 低信頼ログ読み込み
  - FAQ管理へ移動
  - AIデータへ移動

## 画面の役割

- FAQ管理: FAQの追加・編集・評価・改善
- AIデータ: JSON/CSVの取込・出力
- 質問統計: よく聞かれる質問の確認
- 運用: 日常メンテナンスのみ
