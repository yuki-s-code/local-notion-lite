# V155 Chat Priority Layout Fix

## 目的

Local AI Chat の中央チャット領域に FAQ 統計カードと回答モードカードが入り込み、チャットログが見えにくくなる問題を修正しました。

## 修正内容

- 中央チャット本文上部から AI ホームカードを削除
- FAQ件数・承認済み・PDF由来・要確認は右側FAQライブラリ側で確認する設計に統一
- 回答モードは入力欄の横に小さなインラインセレクトとして配置
- チャットログの表示領域を優先
- 小幅画面では回答モードが入力欄下に回り込むレスポンシブ対応を追加

## 方針

中央はチャットに集中し、FAQ状況や管理情報は右側パネル・モーダルに寄せます。
