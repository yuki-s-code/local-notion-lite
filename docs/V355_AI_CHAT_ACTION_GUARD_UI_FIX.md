# v355 AI Chat Action Preview / Grounding Guard Fix

## 修正内容

- AI回答アクションの実行前プレビューを、固定入力欄の上ではなくチャット本文のスクロール領域内に移動。
- プレビュー本文が長い場合もプレビュー内をスクロールでき、実行ボタンが入力フォームに隠れないように修正。
- ワークスペース全体への質問で、ページ文脈も参照候補も0件の場合は生成AIに投げず、根拠不足として停止。
- 根拠0件で小型LLMが一般知識や学習内容を断定する挙動を抑制。
- 生成AIが同じセクションを2回繰り返した場合に、重複セクションを表示前に抑制。

## 意図

右下AIは業務用の根拠付きアシスタントとして使うため、根拠がない場合は無理に回答しない方針にした。
