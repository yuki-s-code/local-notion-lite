# v201 Guided Medium Answer & Rule Merge

## 目的
v200では、カテゴリと提案チップは正しく当たっていても、信頼度が60%未満の場合にFAQ本文を返さないため、
「学童の費用はどのように確認するか」のような短い質問で回答が出ないケースがありました。

v201では、誤回答を防ぐガードは残したまま、カテゴリ・キーワード・汎用ルールが一致している場合は
50%以上で「候補回答」としてFAQ本文を表示するようにしました。

## 主な変更

- 中信頼の閾値を 60% から 50% に調整
- 85%以上のみ高信頼のまま維持
- 50〜84%は「候補回答」として本文を表示
- 学童費用用の標準言い換え辞書を追加
- 学童費用用の標準汎用ヒットルールを追加
- 学童減免用の標準言い換え辞書・汎用ルールを追加
- 既存の synonyms.json / rule-profiles.json がある場合でも、新しい標準IDのみ自動追加入力
- medium回答の表示を「結論: ...」形式に改善

## 期待される動作

質問:

```txt
学童の費用はどのように確認するか
```

期待:

```txt
信頼度 50〜84%
中
おそらく次のFAQが近いです。

結論: 放課後児童クラブの費用は、自治体の公式ページ、募集要項、利用案内、利用決定通知で確認するのが確実です...
```

## 共有ファイルへの反映

既に共有フォルダに以下が存在する場合でも、新規標準IDだけ追加されます。

```txt
smart-assist/synonyms.json
smart-assist/rule-profiles.json
```

追加される主なID:

```txt
syn_afterschool_fee
syn_afterschool_reduction
rule_afterschool_fee
rule_afterschool_reduction
```
