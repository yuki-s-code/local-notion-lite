# v24 Page Open Fix

サイドバーのページタイトルクリック時に、本文ではなくDB/ゴミ箱/別ビューが残ることがある問題を修正しました。

## 修正内容

- メイン表示状態を `empty / page / database / trash` で明示的に管理
- ページを開く処理では必ず `page` モードへ切替
- DBを開く処理では必ず `database` モードへ切替
- ゴミ箱を開く処理では必ず `trash` モードへ切替
- サイドバーのページ行全体をクリック可能に変更
- ページタイトルクリック時にイベント伝播で別処理が混ざらないよう調整

これにより、サイドバーのページをクリックしたときは必ず対象ページ本文が表示されます。
