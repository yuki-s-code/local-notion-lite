# v106 Command Palette and Settings

## 追加内容

- Cmd/Ctrl+K のコマンドパレットを強化
  - ページ、Journal、Inbox、DBに加えて Tasks / Files を検索対象に追加
  - ホーム導線、クイックキャプチャ、Tasks、Files、Links、共有管理、設定を操作に追加
  - ショートカットヒント表示を追加
- 設定画面を追加
  - 表示密度: Comfortable / Compact
  - テーマ: Soft / Light
  - 自動保存間隔
  - Journal初期表示の設定枠
  - 共有フォルダ変更 / 再同期
  - API / SQLite状態表示
- Cmd/Ctrl + , で設定を開けるように追加

## 方針

BlockNote本体には深く触らず、安定しているv105の周辺UIとして追加しています。
