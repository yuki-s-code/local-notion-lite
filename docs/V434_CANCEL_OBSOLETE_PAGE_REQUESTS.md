# V434: 不要になったページ読込みの中断

## 対象

- 通常のページ切替
- リンク先プレビューの切替

## 変更内容

ページを続けて選択した場合、前のページ取得リクエストを `AbortController` で中断します。

- 新しいページを選ぶと、前の `GET /pages/:id` を中断
- 新しいリンク先プレビューを選ぶと、前のプレビュー取得を中断
- 中断はエラー表示にしない
- 既存の NavigationCoordinator による古い結果の表示防止は維持

## 効果

低速な共有フォルダ・混雑時でも、利用者が既に移動した先の読込み結果を待たず、現在選択中のページの表示にリソースを寄せられます。

この変更は本文・タグ・履歴・保存形式を変更しません。
