# V358 Smart Chat Tone

右下AIチャットの回答が硬い行政文書風になりやすかったため、回答スタイルをチャット向けに調整しました。

## 変更点

- 標準回答モードの表示名を「スマート」に変更
- 標準回答の指示を「結論・根拠・確認事項」固定から、自然な会話調へ変更
- ページ要約は、最初に「つまり何のページか」を短く示し、その後に重要点を整理
- TODO抽出は、実際に次に動ける表現に変更
- 「確認事項」を毎回固定で出すのではなく、必要な場合だけ短く表示
- 生成AIプロンプトに、行政文書のような硬い言い回しや同じ見出しの反復を避ける指示を追加
- 検索ベースfallback回答も少し自然な表現へ変更

## 維持したこと

- 根拠なし生成の停止
- 参照元固定/除外
- ページ文脈の利用
- llama-server常駐/1回起動切替
- AI回答アクション
- 履歴保存
