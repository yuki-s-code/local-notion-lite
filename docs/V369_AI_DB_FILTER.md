# V369 AI DB Filter

AIチャットがデータベース行を単にSemantic検索するだけでなく、自然言語の条件からDB行候補を補強するための改善です。

## 追加内容

- 「未完了」「未対応」「完了」「対応中」「確認待ち」などのステータス条件を検出
- 「今日」「今週」「今月」「期限切れ」「期限が近い」などの日付条件を検出
- 「5000円以上」「1000以下」などの数値条件を検出
- 条件に合うDB行を `database_row` の根拠候補として生成AI回答に合流
- 回答ステータスに `DB条件 ○件` を表示
- 根拠選定理由に `DB条件抽出を使用` を表示

## 方針

LLMにSQLを生成させず、コード側の安全な条件判定で候補を抽出します。これにより、ローカル環境でも安定しやすく、誤ったSQLや危険なクエリを避けられます。
