# v409 Tag suggestion feedback

Tag suggestions remain opt-in. When a suggestion is added, the tag receives a small local ranking boost. When it is dismissed, it is hidden for the current page view; repeated dismissals make it less likely to appear later.

Feedback is saved only in the local renderer `localStorage` under `local-notion:tag-suggestion-feedback:v1`. It is not written to shared workspace files, does not affect other users, and never causes automatic tag assignment.
