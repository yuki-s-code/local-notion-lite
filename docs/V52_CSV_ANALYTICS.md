# v52 CSV / Analytics

## 追加内容

- データベースのCSV保存
- 現在のビューだけをCSV保存
- 全件CSV保存
- CSV読み込み
- CSVヘッダーからプロパティを作成
- 値から列型を簡易推定
- データ分析パネル
  - 数値列の平均・合計・最小・最大
  - チェック列の完了率
  - 日付列の最古・最新
  - select/text列の上位分類

## 方針

新しい重いUIライブラリはまだ追加せず、既存の安定構成のままCSVと分析機能を追加しています。
今後、さらに本格化する場合は TanStack Table / Recharts / Tremor などの導入を検討できます。
