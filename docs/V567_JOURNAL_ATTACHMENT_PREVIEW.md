# v567 — Journal 添付プレビュー

## 方針
- Journal 添付はすでに専用フォルダに保存・削除バックアップ・復元される。
- OCR キューは Inbox 専用の共有フォルダ排他制御であり、Journal へ流用すると処理所有権と状態保存が二重になる。
- そのためこの版では OCR 機能を新設せず、先に安全な画像/PDFプレビューを追加する。

## 追加内容
- 添付一覧を視認しやすいカード表示へ変更。
- 画像/PDFをクリックするとアプリ内モーダルでプレビュー。
- 他形式は既定アプリ／ブラウザで開く。
- すべての添付に別ウィンドウで開く操作とダウンロード導線。
- ジャーナル切替時はプレビューを自動で閉じる。

## 影響範囲
- 既存の共有フォルダ添付保存先・Journal削除/復元・本文BlockNote添付URLには変更なし。
