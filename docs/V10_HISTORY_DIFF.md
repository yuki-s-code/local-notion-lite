# v10 History Preview / Diff

このバージョンではページ履歴の扱いを強化しました。

## 追加機能

- 履歴バックアップの本文プレビュー
- 現在ページと履歴バックアップの行単位差分表示
- 復元前の内容確認
- 復元後は現在内容を自動バックアップしてから上書き

## 方針

共有フォルダ運用では、誤保存・競合・ネットワーク断から復旧できることが重要です。
v10では、復元前に「何を戻すのか」を確認できるUIを追加しています。

## 注意

差分はMarkdownミラー `content.md` を対象にした行単位差分です。
BlockSuite/Yjsの構造差分ではありません。
