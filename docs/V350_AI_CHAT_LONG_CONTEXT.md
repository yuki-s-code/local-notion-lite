# V350 AI Chat Long Context

右下AIチャットの読み込み量と回答長を切り替えられるようにした版です。

## 変更点

- 右下AIチャットに「読込」切替を追加
  - 高速: 約2,400文字
  - 標準: 約6,000文字
  - 詳細: 約12,000文字
- 右下AIチャットに「長さ」切替を追加
  - 短め
  - 標準
  - 詳しく
- ページ本文をチャンク化してプロンプトに渡すように変更
- 「このページを要約」「TODO抽出」で、読み込んだ文字数・チャンク数・ctx・maxを画面に表示
- 会社PC高速モードでも、明示的に標準/詳細を選んだ場合は右下AIチャットだけ長めの生成を許可
- 生成AIが使えない場合のページ本文fallbackも、読み込み量を表示

## 注意

詳細モードは生成時間とメモリ使用量が増えます。会社端末では、まず「標準」から確認してください。
