# V780 Web Platform Suite

V779を基準にWeb Builderを制作・データ・運用まで一貫したWeb Platformへ拡張。

## 追加
- Live Binding / Smart Variables
- CMS collection rendering
- Shared Components
- Responsive rules editor
- Form builder runtime and validation
- AI layout safe optimizer
- Version difference summary
- Design presets and tokens
- Publish provider settings and deploy guide
- Export runtime unification

## 効率化
- PreviewとExportは同一runtimeを使用
- データ変換はcompilerで一度だけ実行
- レスポンシブ規則はCSS生成のみ
- AI最適化は明示操作のみ
- 外部公開サービスの認証情報は保存しない
- 既存プロジェクトはnormalizeWebProjectで後方互換移行
