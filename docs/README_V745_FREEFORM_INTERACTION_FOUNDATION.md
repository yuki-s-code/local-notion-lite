# V745 Freeform Interaction Foundation

## 変更内容

- 選択ツールの空白ドラッグを矩形範囲選択へ変更
- Shift / Command / Ctrlを押しながらの範囲選択で既存選択へ追加
- 手のひらツールのドラッグをキャンバス移動へ分離
- Spaceキーを押している間、一時的にキャンバス移動を有効化
- Spaceパン中のカーソル表示を追加
- カードや画像の上からでもペン・直線描画を開始可能に変更
- 描画完了後に新しい線を自動選択しないよう変更
- Pointer Cancel時にも描画・ドラッグ・選択処理を安全に終了
- 選択矩形の視覚表示を追加

## 操作

- 選択ツール + 空白ドラッグ: 範囲選択
- Shift / Command / Ctrl + 範囲選択: 選択へ追加
- 手のひらツール + ドラッグ: キャンバス移動
- Space + ドラッグ: 一時的にキャンバス移動
- ペン / 直線ツール: カード上からでも描画開始可能
