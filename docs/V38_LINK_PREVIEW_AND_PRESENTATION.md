# v38 Link preview and presentation controls

## Changes

- Fixed card insertion leaving the typed `@` behind.
- Added conversion controls for already-created page links: inline / card.
- Clicking a page link in the BlockNote body now opens a Notion-like right preview drawer instead of relying only on direct navigation.
- The preview drawer includes a button to open the linked page as the main page.
- Existing legacy link formats are still normalized on load.

## Link roles

- `@` is for linking existing pages.
- Candidate menu supports inline link and card link insertion.
- Existing links can be converted from the editor link display tools.
