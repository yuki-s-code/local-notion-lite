# V688: 選択範囲AI編集の近接ポップオーバー化

## 目的
BlockNote標準の書式フローティングと独自AIボタンの重なり、およびAI編集操作後にエディタ先頭まで戻る操作負荷を解消する。

## 変更
- 選択範囲のAI起点は、`.bn-formatting-toolbar` の実測矩形を避けて配置する。
- 選択後のAI編集は、選択範囲近くのPortalポップオーバーで実行・プレビュー・置換まで完結する。
- 選択時のツールバー位置を正確に測るため、selectionchangeの次フレームで位置計算する。
- ページ全体を対象としたAI編集だけは、従来どおりエディタ上部の静的パネルを用いる。

## 性能
- 選択時はDOM矩形の測定のみで、検索・索引・OCR・Semantic更新・ネットワーク要求は行わない。
- 生成要求は従来と同じ明示操作時だけ発生する。
