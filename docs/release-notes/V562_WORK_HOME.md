# V562 作業ホーム画面

## 方針

既存の `HomeDashboard` を置き換えず、ホーム画面を「今日の作業を再開する入口」として再設計しました。共有フォルダのデータは増やさず、既存のDashboardサマリーと端末に保存済みのワークスペースタブ情報だけを利用します。

## 追加・改善

- 優先確認：期限切れ、今日が期限、共有競合を一目で表示
- 作業を再開：ローカルに保存されているワークスペースタブを最大6件表示
- 最近の更新：ページに加え、最近更新されたデータベースを追加
- クイックアクセス：Inbox、Tasks、添付、リンク管理へ直接移動
- Journal：直近の記録または今日のJournal作成を表示
- 既存のDashboardキャッシュが旧形式の場合、一度だけ再生成してDB一覧を表示

## UI

- 大きな今日の入口、状況カード、二列レイアウト
- 期限・競合は控えめな警告色で視認性を確保
- 画面幅に応じて2列・1列へ自動再配置
- 小さい画面では横幅を取りすぎず、カード内容を縦配置

## データへの影響

- ページ・データベース・共有フォルダデータの形式変更なし
- タブの再開一覧は `localStorage` の既存ワークスペース設定を読むだけ
- Dashboard APIに軽量な `recentDatabases` を追加
