# v512.2 Read-only BlockNote workbench

## 実装内容

- 比較表示・分割表示の参照ペインを、Markdownのプレーンテキスト表示から **BlockNote閲覧専用表示** に変更。
- 参照ペインごとに独立したBlockNoteインスタンスを作成。編集画面のエディターは共有しない。
- 参照ペインでは `editing={false}` とし、保存・自動保存・履歴・AI編集・DB編集を実行しない。
- 参照ペイン内のページリンクは通常どおり開ける。
- 保存済みページの `updatedAt` をキーへ含め、元ページを保存した後に古いエディター状態を再利用しない。

## 共有フォルダの安全性改善

- ページ保存時の競合検出を修正。編集開始時点の `baseUpdatedAt` と共有フォルダ側の更新日時が異なる場合は、ローカルSQLiteの同期状態に関係なく競合として退避する。
- 共有JSON更新用ロックの期限を30秒から120秒へ、ロック取得待ちを4秒から12秒へ変更。SMB/NASが遅い環境での誤った期限切れ扱いを減らす。

## 制約

- これは閲覧専用の参照ペインであり、CRDTを用いたリアルタイム共同編集ではない。
- 共有フォルダだけでの同時編集は引き続き対象外。主編集画面は1ページにつき1人が編集する運用を前提とする。
