# V671 — ページの「今どこ？」と関係の物語表示

## 追加した機能

- ページ本文の上部に、親子階層・状態・タグを表示する「今どこ？」バーを追加。
- 「関係を物語で見る」を開いた時だけ、既存のローカル知識グラフを取得。
- 関係はAIや本文の再走査を使わず、SQLiteの `workspace_link_index`、ページ親子、タグ、DB行参照から決定的に文章化。
- 親・参照元・リンク先・子ページ・共通タグを、クリック可能な関係チップとして表示。
- マップへの導線を同じバーに配置。

## 性能方針

- 初期ページ表示ではネットワーク/SQLite追加読み込みをしない。
- グラフ取得は物語パネルを開いたときだけ。
- 1ページ48ノードに制限し、クライアントメモリ内に最大24ページ・45秒TTLでキャッシュ。
- ページ本文、OCR、ベクトル検索、共有フォルダ全走査は行わない。
