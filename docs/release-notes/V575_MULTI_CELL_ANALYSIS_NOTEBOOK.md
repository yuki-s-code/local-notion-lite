# V575 分析ノートブック：複数セル化

## 目的

分析ノートブックを、単一のSQL欄からJupyterLabに近い複数セル形式へ拡張しました。

## 追加内容

- SQLセルを複数追加し、個別に実行できる。
- メモセルを追加し、分析の前提・根拠・結論をノートと一緒に保存できる。
- 各SQLセルに独立した表、グラフ、CSV出力を持たせる。
- セルは上下移動・削除できる。
- SQLの末尾に付ける通常のセミコロン `;` を許可する。
- ただし、複数SQL文、更新SQL、拡張機能操作は引き続き禁止する。

## 互換性

- 既存の単一SQLノートを開くと、SQLセル1つのノートとして自動変換する。
- `analysis_notebooks.cells_json` を端末側SQLiteに追加する。
- 既存の `sql_text` / `chart_json` は残し、旧ノートとの互換性を保つ。

## 保存先

- 分析ノート：ローカルSQLite (`local.sqlite`)
- DuckDB分析キャッシュ：ローカル `analysis.duckdb`
- 共有フォルダのページ、DB、Journal、OCR正本は更新しない。
