# v634 分析セルの再描画分離

- 分析ノートの各セルを `AnalysisNotebookCellCard` として独立・memo化。
- セル編集時は、対象セルのオブジェクトだけを置換し、他セルの入力UIを再描画しない。
- 上流セルの候補リストはセルごとに署名付きでキャッシュし、無関係な編集で `select` の候補を再生成しない。
- 実行・スナップショット・ダッシュボード固定は安定したコールバック経由にし、親画面の状態更新による子セルの不要な更新を防止。
- 前版の仮想結果表・ページキャッシュと組み合わせ、セル数・結果行数が多いノートでの操作感を改善。

## 確認

- TypeScriptの単体構文変換を実施。
- プロジェクト全体の `tsc --noEmit` は、添付ZIPに `node_modules` と lockfile が無いため Electron / Node 型定義解決の開始段階で停止する。
