# v609 サブアイテム行番号位置ずれ修正

## 原因
サブアイテムの開閉ボタン、子階層のインデント、進捗バッジを行番号セルへ混在させていたため、行番号が親子の深さ・子件数によって横方向に移動していました。

## 修正
- 行番号セルは番号のみを固定幅・中央配置で表示します。
- 親子の開閉、インデント、進捗は独立した「階層」列へ移動しました。
- サブアイテム未使用のDBには階層列を出しません。
- 仮想スクロールのpadding行と集計フッターの列数を、階層列の有無に応じて調整しました。

この変更は表示専用であり、DB行、Relation、サブアイテム、Index、localStorageの折りたたみ状態に影響しません。
