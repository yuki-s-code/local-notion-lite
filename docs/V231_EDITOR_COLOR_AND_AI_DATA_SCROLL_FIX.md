# V231 FAQ Editor Color and AI Data Scroll Fix

## 修正内容

- FAQ Editor の「FAQ Editor / 新しいFAQを作成」の文字色が背景と反転して見にくくなる問題を修正。
- FAQ Editor ヘッダーの文字色、ラベル、閉じるボタンを明示的に上書き。
- AIデータ画面がスクロールできず、下のカードやボタンが見えなくなる問題を修正。
- Smart Assist 管理画面モーダルを flex レイアウト化し、ヘッダー・タブを固定、各タブ本文だけをスクロール可能に変更。
- AIデータカード内の JSONサンプル / CSV列表示が横にはみ出さないように折り返しを追加。

## 目的

FAQ編集とAIデータ管理を、画面サイズに関係なく読める・操作できる状態にする。
