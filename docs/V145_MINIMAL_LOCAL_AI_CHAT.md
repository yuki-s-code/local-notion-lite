# v145 Minimal Local AI Chat

## 目的
Local Generative Assist の画面全体が縦に伸び、ページ全体スクロールが必要になっていたため、チャット中心の固定レイアウトへ調整しました。

## 変更内容
- ページ全体を `100vh` 固定レイアウトに変更
- スクロール領域をチャットログと右サイドバーだけに分離
- 上部ヘッダーをコンパクト化
- 「質問 / 根拠 / 育成」ガイドを短いピルUIに整理
- クイック質問・FAQ操作ボタンを横スクロールのコンパクトUIへ変更
- 右サイドバーはFAQライブラリと参照元を優先表示
- ナレッジベース、品質、候補などは通常表示では隠し、詳細ツール側へ寄せる前提に変更

## 狙い
通常利用では、中央のチャットと右側のFAQ/根拠だけを見れば操作できるようにしています。
