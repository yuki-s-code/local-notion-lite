# V360 AI Chat Intent Router / Grounding Control

右下AIチャットの回答品質を上げるため、回答前に質問意図を判定するルーターを追加しました。

## 追加内容

- 質問意図の自動判定
  - このページの要約
  - このページのTODO抽出
  - このページの関連情報
  - 続きの質問
  - 手順・方法
  - 比較・整理
  - FAQ回答
  - 文面作成
  - 広い質問/曖昧な質問
  - 通常のワークスペース質問

- 検索語の内部補正
  - 学童 → 放課後児童クラブ / 入会 / 費用 / 減免 / 延長 / 長期休業 / 見学予約
  - 料金 → 費用 / 利用料 / 月額 / おやつ代 / 延長利用料
  - 申請 → 申請方法 / 手続き / 提出先 / 期限 / 必要書類

- 根拠スコア制御
  - 高: 生成AI回答可
  - 中: 生成AI回答可。ただし根拠表示
  - 低: 広い質問では聞き返し優先
  - なし: 生成AIへ投げない

- 聞き返し制御
  - 「芦屋市について教えて」のような広い質問では、候補を見せながら確認する
  - 根拠が弱い場合は断定しない

- 右下AI UI
  - 「判断: 質問意図 / 検索方針」を表示
  - 根拠スコアを表示

## 目的

小型ローカルLLMは、根拠が弱い状態で生成させるとそれらしい断定を作りやすい。V360では、回答前に「答えるべきか / 聞き返すべきか / どの検索を使うべきか」を判断して、より安全で適切なチャット回答に寄せています。
