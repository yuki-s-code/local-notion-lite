# v175 FAQ Intent Ranking Fix

## 目的
v174 では `kuromoji.js + Fuse.js` の検索自体は動作したが、「学童の就労条件は？」のような質問で、
「放課後児童クラブの概要」FAQが高スコアになり、具体的な就労要件FAQより上に来ることがあった。

## 修正内容
- `japaneseFaqSearch.ts` に質問意図判定を追加
- 「就労・仕事・勤務」+「条件・要件・基準・週何日・何時」系の質問を就労要件意図として判定
- 就労要件意図では、以下を含むFAQを加点
  - 就労要件
  - 勤務終了時間
  - 午後4時 / 16時
  - 週3日
  - 3か月
  - 継続勤務
  - 通勤時間
- 概要FAQ・キッズスクエア比較FAQなど、一般説明FAQを減点
- 一致理由に「就労条件の意図に一致」を追加

## 期待される挙動
「学童の就労条件は？」では、概要FAQではなく、
「勤務時間が何時までなら入れる？」「週何日働いていたら学童に申し込める？」等の就労要件FAQが上位に来る。
