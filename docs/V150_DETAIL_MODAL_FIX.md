# v150 Detail Modal Fix

## 目的
Local Generative Assistの「詳細を開く」をインライン展開からモーダル表示へ変更し、固定レイアウト内でスクロール不能になる問題を防ぐ。

## 変更点
- 詳細ツールを中央モーダル化
- モーダル上部と下部に閉じる導線を追加
- モーダル本文だけをスクロール可能に変更
- スマート検索、重要文、トークン、候補診断をモーダル内カードとして整理
- 小幅画面では1列表示に切替

## 効果
- 詳細画面を開いてもチャット画面のレイアウトを押し下げない
- 詳細画面内でスクロールできる
- 閉じる操作が常に見つけやすい
