# v314 FAQ Improvement Apply Workflow

## Purpose
v314 turns the generated FAQ improvement draft into an operational workflow.

## Changes
- Generated FAQ improvement draft can be edited before applying.
- Improvement preview fields are editable: improved question, improved answer, likely questions, paraphrases, negative terms, and suggested actions.
- Added simple quality checks for:
  - question rewritten or not
  - answer rewritten or not
  - likely question count
  - duplicate likely questions
  - paraphrase count
  - possible date/money additions not present in the original answer
- Added one-click save actions:
  - save search hints only
  - save full question/answer and search hints
- Added backup metadata on the FAQ record before applying generated changes.

## Notes
The generated content is still treated as a draft. Users should confirm facts before saving, especially dates, amounts, deadlines, and procedure names.
