# V134 Local Smart Assist

OllamaやクラウドAIを使わず、完全ローカル・軽量構成で「AI風」の整理支援を追加しました。

## 追加内容

- Local Smart Assist 画面を追加
- Fuse.js による曖昧検索
- 日本語向けの軽量トークン化
  - NFKC正規化
  - カタカナ→ひらがな寄せ
  - 漢字/かな/英数字抽出
  - 2-gram / 3-gram による部分一致補助
- 辞書ベースのタグ候補
- ルールベースTODO抽出
- 重要文抽出
- Relation候補スコアリング
- 重複候補検出
- 未整理/空欄診断

## 使い方

左サイドバーの 🧠 ボタンから開きます。

検索例：

- 学童 期限
- 会計年度 休暇
- PDF 未整理
- Relation候補
- 予算 契約

## 設計方針

この版ではAIモデルを同梱しません。会社PCの容量を圧迫しないよう、次の軽量技術で構成しています。

- Fuse.js
- 独自軽量トークナイザー
- キーワード辞書
- ルールベース抽出
- 類似度スコアリング

将来的に小型Embeddingや外部ローカルAIエンジンを足す場合も、Local Smart Assistの画面を拡張できます。
