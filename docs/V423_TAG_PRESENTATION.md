# V423 タグの色・グループ化

## 追加内容

- タグごとに表示グループを設定できます。
  - 業務分野 / 年度 / 対象者 / 状態 / その他
- タグごとに色を設定できます。
  - グレー / 青 / 水色 / 緑 / 黄 / 橙 / 赤 / 紫 / 桃
- 設定は `workspace/tag-presentation.json` に保存され、同じ共有フォルダを開く端末で共通です。
- タグ名、ページ本文、ページのタグ値、ページ履歴は変更しません。

## 操作

1. タグ管理画面でタグを選択します。
2. 「分類・色」でグループと色を選択します。
3. 画面上は即時反映し、共有フォルダへの保存は短い待機後にまとめて実行されます。

## 表示範囲

- タグ管理の一覧
- タグ管理の選択詳細
- ページプロパティの既存タグチップ

## 保存データ

```json
{
  "version": 1,
  "revision": 1,
  "settings": {
    "放課後児童クラブ": { "group": "業務分野", "color": "blue" },
    "令和8年度": { "group": "年度", "color": "purple" }
  }
}
```

無効なグループ・色・過大な辞書はサーバー側で除外します。
