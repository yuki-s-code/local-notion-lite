# v146 Responsive Iconic Local AI Chat

## 目的
ローカルAIチャット画面を、画面サイズに応じて見やすく変化するレスポンシブUIへ改善しました。

## 主な変更

1. **レスポンシブ改善**
   - デスクトップ: 中央チャット + 右FAQ/根拠パネル
   - 中幅画面: チャット上段 + FAQ/根拠パネル下段
   - 小幅画面: チャット優先、補助情報はコンパクト表示

2. **画面全体スクロール抑制**
   - ページ全体ではなく、チャットログとサイドパネルを個別スクロールにしました。
   - 画面全体が縦に伸びて見失う問題を軽減しました。

3. **アイコン活用**
   - FAQ、根拠、回答品質、JSON取込、FAQ化などの操作にアイコンを追加しました。
   - 文字だけのUIより、ユーザーが機能を把握しやすくなるようにしています。

4. **JSON取込UI改善**
   - JSON取込を下に伸びる表示ではなく、中央モーダルとして表示します。
   - JSON貼り付け欄を広くし、スマホ幅でも見やすいレイアウトにしました。

## 備考
機能ロジックはv145を維持し、UI/UX改善を中心にしています。
