# v40 Notion Preview and BlockNote Layout

## 変更内容

- 右サイドバーのリンク先プレビューを、タイトルだけでなく本文まで表示するページプレビューに変更。
- プレビュー本文もBlockNoteで描画し、メイン本文とプレビュー本文を別インスタンスとして扱うMulti-Editor構成に変更。
- プレビュー内のリンクもクリック可能で、さらに右サイドバー内で別ページをプレビュー可能。
- BlockNoteの標準ルールに寄せ、本文内を直接DOM改変せず、保存済みBlockNote blocksを読み取り専用で描画。
- スラッシュメニューに「2カラム」下書き挿入を追加。現時点ではBlockNoteの標準ブロックとして安全に挿入し、将来的なMulti-Column Blocks正式対応へ置き換えやすい構成。

## 方針

エディタ本体はBlockNoteのAPIに沿って操作し、独自DOMで無理に本文を差し替えない。共有フォルダ保存・履歴・ロック・バックリンクの既存設計は維持する。
