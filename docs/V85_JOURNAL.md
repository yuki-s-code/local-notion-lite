# V85 Journal

Adds a daily journal feature separate from regular pages.

- Sidebar journal card
- Today / previous / next day navigation
- Date-based journal files under `journals/YYYY-MM-DD/journal.json`
- BlockNote-based daily memo editor
- Mood, weather, and tags metadata
- Autosave with shared-folder storage

The feature intentionally avoids page locks for journals in this first implementation. It is designed as a personal daily note layer and can be extended with locks later if multiple users edit the same date.
