# V177 Abstract Query Intent Search

曖昧な言葉・抽象的な言葉でもFAQ検索が外れにくくなるように、検索エンジンと回答整形を改善しました。

## 追加内容

- 「できる」「いける」「大丈夫」「使える」などの可能性確認を検出
- 「どうしたらいい」「やり方」「流れ」などの手順質問を検出
- 「誰が」「対象」「条件」「要件」などの対象・資格質問を検出
- 「困った」「わからない」「相談」などのヘルプ質問を検出
- 抽象語を具体カテゴリへ寄せる同義語辞書を追加
- 概要FAQに寄りすぎないようにランキングを補正
- 回答側で「対象者・条件・手続き」の観点に整理

## 期待される質問例

- 学童使える？
- これっていける？
- どうしたらいい？
- 対象になる？
- 何が必要？
- 仕事してたら入れる？
- よくわからないので教えて

## 狙い

完全な生成AIを使わず、ローカルFAQ検索のまま、ユーザーが自然に聞いた短い質問や曖昧な質問に強くします。
