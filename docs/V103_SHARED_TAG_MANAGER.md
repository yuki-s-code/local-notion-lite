# V103 Shared Tag Manager

- Fixed page property tag editing so multiple tags can be entered reliably.
- Replaced comma-only controlled input with a chip-based tag input.
- Tags can be added with Enter, comma, blur, or by selecting existing tag suggestions.
- Existing tags are collected from pages, journals, inbox items, and database select/multi-select values.
- Added a compact tag manager inside page properties to reuse existing tags.
- Applied the same tag input to the editable side peek panel.

This avoids the old issue where typing a comma immediately normalized the value and made only the first tag easy to enter.
