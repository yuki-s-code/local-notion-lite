# v20 Notion風UIと読みやすいリンク

## 変更点

- `@[[タイトル|id]]` を新規挿入では使わず、本文上は `📄 ページタイトル` と表示するように変更。
- 旧形式 `@[[タイトル|id]]` は引き続き読み取り可能。
- データベース埋め込みも新規挿入では `🗃️ データベースタイトル` と表示。
- 旧形式 `{{database:id}}` は引き続き読み取り可能。
- 画面全体の余白、カバー、カード、テーブル、候補パネルをNotion風に調整。

## 注意

現時点ではページリンクはタイトルで解決します。重複タイトルが多い運用では、次版でBlockNoteカスタムブロック化してIDを内部保持する予定です。
