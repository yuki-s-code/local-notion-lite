# v519 — サイドバー起点のデータベースタブ

## 変更
- サイドバーのデータベース名を通常クリックすると、BlockNoteページと共通の作業スペースタブ列へDB参照タブを追加します。
- 同一DBがすでに開いている場合は重複タブを作らず、そのタブを参照先として選択します。
- 通常クリックは右側の閲覧専用参照ペインを開きます。本文ページは編集状態のまま維持されます。
- DBを右クリックすると、比較表示・既存の編集画面を選択できます。
- `＋` からDBを選ぶ導線は削除しました。DBを探す入口はサイドバーに統一しています。

## 安全性
DB参照タブは `DatabaseTable` の `editing={false}` を使用します。DB保存、自動保存、履歴作成、共有フォルダの編集ロック取得は行いません。編集が必要な場合だけ右クリックの「編集画面を開く」から既存の保存・競合制御を利用します。
