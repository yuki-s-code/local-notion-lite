# v210 Operation Progress & Duplicate-safe Import

## 目的
Smart Assist 運用パネルで、重い処理が実行中なのか、いつ完了したのか、何件処理したのかをユーザーが判断できるようにしました。

## 追加内容

- 運用パネルに処理ステータスカードを追加
- 実行中はローディングドットとインジケーターを表示
- 完了時は完了時刻と処理件数を表示
- 失敗時はエラー状態として表示
- FAQ JSON取込中は取込ボタンを無効化
- 再インデックス、node-nlp再学習、低信頼ログ読込、FAQテストの同時実行を抑制
- FAQ JSON取込時に重複FAQをスキップ
- サーバー保存時も同一ID・同一質問の重複を除外

## 重複判定

FAQ JSON取込時は以下を重複として扱います。

- 既存FAQと同じ `id`
- 既存FAQと同じ正規化済み `question`
- 取込JSON内で重複している `id`
- 取込JSON内で重複している正規化済み `question`

重複は上書きせず、スキップします。

## 表示例

```txt
FAQ JSON取込中
新規 7件を保存中...（重複 2件はスキップ）

FAQ JSON取込完了: 新規 7件 / 重複スキップ 2件 ・ 完了 15:32:10
```

## 注意

既存FAQを更新したい場合は、同じ質問を再取込するのではなく、FAQ管理画面で対象FAQを編集してください。
