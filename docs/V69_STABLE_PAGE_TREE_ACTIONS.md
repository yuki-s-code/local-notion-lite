# V69 Stable Page Tree Actions

ページツリー行のHover時に表示される ↑ / ↓ / ＋ / T などの操作ボタンが、タイトル列の幅を再計算してタイトルを短く見せる問題を修正しました。

## 変更点

- `.tree-row-actions` をCSS Gridの通常フローから外し、absolute配置へ変更
- ページタイトル列は常に `minmax(0, 1fr)` で固定
- Hover時もタイトル列の幅が変わらないように調整
- 操作ボタンは右端に薄い背景つきで浮かせて表示
- 選択中ページでは背景色に合わせてアクション領域の色も調整

## 目的

Notion風に、ページツリーの視認性を保ったまま、Hover時だけ操作ボタンを控えめに出すための調整です。
