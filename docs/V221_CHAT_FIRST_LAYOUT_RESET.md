# v221 Chat-first Layout Reset

## 目的

v220の画面で、よくある質問チップ、管理ストリップ、右側管理ペイン、回答スタイル操作がチャット本文を圧迫していたため、Smart Assist画面をチャット中心に再整理した。

## 変更点

- 右側のミニ管理ペインをJSXから撤去
- チャット上部に小型トップバーを追加
- 管理画面への導線をトップバーとヘッダーに明示
- よくある質問チップを小型の横スクロールピルに変更
- 回答スタイル・履歴削除・フィードバック操作などの常時表示を非表示化
- 詳細管理は管理画面モーダルへ集約
- チャットログと入力欄を1カラム中心のモダンなカードUIへ調整

## 設計方針

チャット画面は質問と回答に集中させ、FAQ管理・CSV・統計・再生成・評価は管理画面に分離する。
