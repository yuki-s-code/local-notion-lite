# v53 TanStack Table / Recharts

## Added

- `@tanstack/react-table` for the database grid.
- `recharts` for visual analysis.
- Clickable table headers for client-side sorting.
- Recharts-based analysis cards:
  - numeric average bar chart
  - checkbox completion bar chart
  - select/text distribution pie chart
  - date monthly line chart

## Notes

The existing JSON database format is unchanged. CSV import/export is unchanged. This version improves the renderer/UI layer without changing the shared folder storage schema.
