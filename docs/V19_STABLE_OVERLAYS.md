# v19 Stable Suggestion Overlays

## 目的

v18 までは `@` ページ候補や `/database` 候補がエディタ本文の通常フロー内に表示されるため、候補の表示・非表示で画面の縦幅が変わり、視線が動きやすい問題がありました。

v19 では候補UIを本文レイアウトから切り離し、固定オーバーレイとして表示します。

## 変更点

- `@タイトル` 入力中のページ候補を右上固定のオーバーレイ表示に変更
- `/database 名` または `/db 名` 入力中のDB候補を右上固定のオーバーレイ表示に変更
- 候補パネルにヘッダー、件数、説明文を追加
- 候補リストに最大高さとスクロールを設定
- 本文レイアウトの高さが候補表示で変わらないように修正
- 補助検索ボックスの候補表示も影付きポップオーバーとして視認性を改善

## 操作

- 本文で `@` と入力するとページ候補が表示されます。
- 本文で `@会議` のように入力すると候補が絞り込まれます。
- 本文で `/database` または `/db` と入力するとデータベース候補が表示されます。
- 候補クリックで、ページリンクまたはDB埋め込みマーカーが挿入されます。

## 注意

現在の候補挿入は、入力中テキストをその場置換するのではなく、カーソル位置の次のブロックにリンク/DBマーカーを追加します。次バージョンでは、BlockNoteのSuggestion Menu APIを使った本格統合に進めます。
