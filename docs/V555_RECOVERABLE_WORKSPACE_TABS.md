# V555 Recoverable Workspace Tabs

## 目的
作業中のタブを閉じた後でも、迷わず直前の作業へ戻れるようにする。

## 追加内容
- 最大12件の「最近閉じたタブ」をローカル設定に保存。
- `Ctrl / ⌘ + Shift + T` で最後に閉じたタブを復元。
- タブバーの「復元」ボタンからも同じ操作を実行。
- `Ctrl / ⌘ + W` でアクティブタブを閉じる。
- `Ctrl / ⌘ + Tab` と `Ctrl / ⌘ + Shift + Tab` で前後のタブを移動。
- テキスト入力、BlockNote編集、セレクト操作の最中はショートカットを横取りしない。
- 全タブを閉じた空画面にも、最後に閉じたタブを復元する導線を表示。

## 互換性
- 既存のワークスペース保存形式をそのまま読み込める。
- 閉じたDB行子ページも、既存の通常ページツリーに混ぜずに復元対象として扱う。
- 共有フォルダ上のページ・DBデータには書き込まない。UI状態のみをローカルに保存する。
