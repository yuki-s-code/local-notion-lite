# v65 Modern Balanced UI

v65 はアプリ全体の見た目と配置バランスを整えるデザインリファイン版です。

## 変更点

- 全体のデザイン変数を追加
- サイドバー、本文、データベース、右サイドバーのトーンを統一
- 白基調 + 淡いグラデーション + 控えめな影に整理
- ボタン、カード、メニュー、プロパティ、DBパネルの角丸・余白・境界線を統一
- BlockNote本文エリアの余白とタイトルのバランスを改善
- データベース画面のヘッダー、テーブル、プロパティ、Row previewをさらに洗練
- 画面幅に応じたレスポンシブ調整を追加

## 方針

新しい重いUIライブラリは追加せず、現在安定している React / BlockNote / 自作DB構成を維持したまま、CSSデザインシステムで全体をモダン化しています。
