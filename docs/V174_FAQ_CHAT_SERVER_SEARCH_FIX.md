# V174 FAQ Chat Server Search Fix

## 目的

v173で追加した `kuromoji.js + Fuse.js` の高精度FAQ検索エンジンを、FAQライブラリ検索だけでなく、チャット回答にも必ず使うように修正しました。

## 修正内容

- チャット質問時に `/smart-assist/faqs/query` を優先実行
- `status=all` で承認済み・確認済み・下書きも検索対象にする
- サーバー検索結果から直接回答を生成
- 回答内に以下を表示
  - score
  - 一致理由
  - 一致語
  - 検索方式
  - 関連FAQ
- サーバー検索に失敗した場合のみ従来のローカル検索へフォールバック

## 確認用質問

- 就労要件を教えて
- 学童の就労条件は？
- 勤務時間が何時までなら入れる？
- 週何日働いていたら学童に申し込める？
- 15時半に終わるけど通勤で帰れない場合は？
