# v407 Tag suggestion ranking

- Keeps v406's safe suggestion-only behavior: no automatic tag assignment.
- Uses title/body matches as the primary ranking signal.
- Uses how often an existing tag appears in visible workspace pages only as a small tie-breaker.
- Shows usage count for tags used on at least two pages.
- Adds an "上位候補を追加" action that applies the displayed suggestions in one edit.
