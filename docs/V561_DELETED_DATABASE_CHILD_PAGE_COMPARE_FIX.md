# V561 — 削除済みDB子ページを比較候補から除外

## 修正内容

- DB行の子ページをゴミ箱へ移動／削除した直後、比較・分割参照の候補から即座に取り除くように修正。
- 比較画面を開いたまま削除した場合でも、`page-tree-mutated` とDB子ページ削除イベントを受けて候補一覧を再取得。
- DB行本文の更新が共有フォルダ遅延や中断で完了しなかった場合に備え、子ページ削除時はリンク索引の `database-child-page` エントリを先に削除。
- 比較候補取得時に、次の条件を満たさない古い索引を自動除外・掃除。
  - ページが存在する
  - ページがゴミ箱に入っていない
  - 親が対象DB行である

## 影響範囲

- 比較・分割のDB行子ページ候補
- DB行子ページのゴミ箱移動・リンク解除

通常のページ、データベース、DB行、共有フォルダの保存形式は変更しません。
