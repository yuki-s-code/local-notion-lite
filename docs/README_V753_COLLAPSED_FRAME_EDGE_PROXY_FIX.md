# V753 折りたたみフレーム接続線修正

## 修正内容

- 折りたたんだフレーム内の子ノードへ接続していた外部エッジを、親フレームの境界へ代理接続するよう変更。
- 子ノード側に保存されていた接続ハンドル位置は、折りたたみ中のみ解除し、親フレームに対して最適な上下左右の接続点を再計算。
- 同じ折りたたみフレーム内で完結する内部エッジは非表示。
- 展開すると元の子ノードと元の接続ハンドルへ自動復帰。
- 折りたたみフレームの表示高さ58pxを接続点計算にも適用し、非表示の元フレーム高さへ線が接続される問題を防止。
- 入れ子フレームでも、最寄りの折りたたみ親フレームまでたどって代理表示。

## 保守性

折りたたみ表示の計算を `freeformCollapsedProjection.ts` へ分離し、画面コンポーネント内の条件分岐を増やさない構成にしています。
