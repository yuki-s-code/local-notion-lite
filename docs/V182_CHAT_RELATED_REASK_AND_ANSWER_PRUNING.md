# v182 Chat Related Re-ask and Answer Pruning

## 目的
関連FAQをクリックしたときにFAQ管理画面へ遷移せず、関連質問をそのままチャットへ再質問するように変更しました。

また、質問意図と異なるFAQの文が回答本文に混ざる問題を抑制しました。

## 主な変更

- 関連FAQボタンをクリックすると、そのFAQの質問文でチャットに再質問します。
- FAQ管理画面への自動遷移を廃止しました。
- 「子供が熱で休みたい」など子の看護・休暇系の質問では、年休・忌引き・料金・減免系の文を回答本文から除外します。
- 回答本文の重複文を除去します。
- 上位FAQのうち、質問意図に厳密に合うFAQを優先して回答生成します。

## 期待する動作

関連FAQをクリックした場合:

1. ホームへ戻らない
2. FAQ管理画面へ遷移しない
3. その関連FAQの質問でチャット欄に再質問される

例:

- 「子の看護休暇は使えますか？」をクリック
- チャットに同じ質問が送信される
- 回答だけが表示される
