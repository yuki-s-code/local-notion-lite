# v425 Smart Assist: タググループ優先検索

## 目的
タグの表示設定（業務分野／年度／対象者／状態／その他）を、Smart Assistの回答表示だけでなく検索順位にも利用する。

## 動作
1. 質問文・別名辞書から既存タグを抽出する。
2. タグ管理画面で設定されたグループを各タグに付与する。
3. Smart Assistは `tagHints` と `tagHintGroups` を検索APIへ送る。
4. 根拠ページ・FAQのタグが一致すると従来どおり加点する。
5. 一致タグが複数の異なるグループにまたがる場合、最大12点を追加加点する。

例: `放課後児童クラブ`（業務分野）と `夏休み`（年度/状態等として設定）に両方一致する根拠は、片方だけ一致する根拠より優先されやすくなる。

## 安全性
- グループ設定が未設定のタグは従来どおりに動作する。
- タグ名、ページ本文、ページ履歴、FAQ本文は変更しない。
- グループは検索の加点材料であり、タグが質問に明示的に一致した場合だけ使用する。質問文だけから新しいタグを推測して付与・検索対象化しない。
