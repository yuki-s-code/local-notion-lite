# V353 AI Chat Readability

右下AIチャットの設定項目増加によりチャット領域が狭くなっていたため、UIを整理した。

## 変更

- 右下AIドロワーをチャット時だけ横幅760pxに拡大
- 設定項目をデフォルト折りたたみに変更
- 上部に現在設定のステータスチップを表示
- llama-server診断を設定内の詳細パネルへ移動
- 読込量プリセットの目安を設定内にカード表示
- クイックアクションをコンパクト化
- チャット本文領域の最小高さと本文行間を改善
- 入力欄を下部に固定

## 現在の読込量目安

- 高速: 約2,400文字 / ctx 2048
- 標準: 約6,000文字 / ctx 4096
- 詳細: 約12,000文字 / ctx 6144以上

## 精度の考え方

ページ要約やTODO抽出は、読み込まれた本文範囲に含まれる情報だけを根拠にする。
読み込み範囲外の本文は回答に反映されないため、長文では「詳細」か、将来的な全チャンク段階要約が必要。
