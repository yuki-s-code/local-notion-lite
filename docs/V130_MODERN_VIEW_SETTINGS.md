# v130 Modern View Settings

## 目的

データベースのビュー設定画面を、Notion風のモダンなカードUIへ刷新しました。

## 変更点

- ビュー設定パネルをヒーローエリア付きのカードUIに変更
- 現在のビュー名・ビュータイプ・行数・表示列数を上部に集約
- Table / Board / Calendar / Gallery / Timeline / Gantt をカード型ボタンで選択可能に変更
- Boardのグループ列、Calendar/Timeline/Ganttの日付列を設定カード内に整理
- 高度フィルターを独立したカードとして表示
- ソート設定を独立したカードとして表示
- 条件未設定時の空状態メッセージを追加
- レスポンシブ対応を追加

## 既存機能への影響

保存形式、Filter/Sortのロジック、ビュー保存ロジックは変更していません。
UIを中心に改善しているため、v129のDBデータはそのまま利用できます。
