# v137 Chat-first Local Smart Assist

## 目的

v136 の Local Smart Assist は検索、FAQ候補、重要文、トークン、候補診断が同時に表示され、情報量が多くなりすぎていた。v137 では、日常利用を **FAQ Chat中心** に整理し、必要な時だけ詳細ツールを開く構成に変更した。

## 変更内容

- オフライン整理アシスト画面をチャット中心UIに刷新
- 主要導線を「質問 → 根拠付き回答 → 参照元を開く」に集約
- 根拠・候補パネルを右側に配置
- 自動FAQ候補を小さなクイック質問として表示
- 詳細検索、重要文抽出、トークン表示、候補診断は折りたたみ式の「詳細ツール」へ移動
- 既存の Fuse.js / 日本語トークン / 類義語 / TODO抽出 / Relation候補 / 重複検出ロジックは維持

## 使い方

1. 左サイドバーの 🧠 Local Smart Assist を開く
2. 中央のチャット欄に質問を入力
3. 回答下の根拠ボタンまたは右側の根拠リストから元情報を開く
4. 詳細な検索や診断が必要な場合だけ「詳細ツールを開く」を押す

## 方針

OllamaやクラウドAIを使わず、会社PCでも軽く使えるように、画面の主役をチャットに絞った。生成AIではなく、ローカル検索・抽出・ルールベースによる根拠付き回答である。
