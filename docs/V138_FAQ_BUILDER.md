# v138 FAQ Builder 基盤

## 目的
Local Smart Assist を「検索チャット」から「育てるFAQ」へ拡張しました。

## 追加内容
- FAQ Builder パネル
- 手動FAQ作成
- 自動FAQ候補の保存
- 選択中ドキュメントからFAQ化
- draft / reviewed / approved / hidden の状態管理
- reviewed / approved FAQをFAQ Chatの回答で優先利用
- localStorage保存による完全ローカル運用

## 方針
PDF FAQは次段階で追加します。v138ではFAQ DBの土台を先に安定化しました。
