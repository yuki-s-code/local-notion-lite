# v200 Conversational Suggestions

## 目的

回答後に「次にできます」「関連して聞けること」「絞り込み」を表示し、ChatGPT / Notion AI のように次の操作へ自然につなげるための更新です。

## 追加内容

- Smart Assist 回答レスポンスに `suggestedActions` / `nextQuestions` / `clarificationChips` を追加
- FAQ JSON の `suggestedActions` / `nextQuestions` を保存・インポート対象に追加
- 回答カードに次アクション、関連質問、絞り込みチップを表示
- チップクリックでそのまま再質問、またはFAQ化・未回答FAQ化を実行
- 費用、申請、休暇、給与などの汎用語から次アクションを自動生成
- サンプルFAQに学童費用・有給開始の提案チップを追加

## FAQ JSON 例

```json
{
  "question": "放課後児童クラブの費用はどのように確認すればよいですか？",
  "answer": "自治体の公式ページ、募集要項、利用案内、利用決定通知で確認してください。",
  "category": "放課後児童クラブ",
  "suggestedActions": [
    "月額利用料を確認する",
    "延長利用料を確認する",
    "減免制度を確認する",
    "担当課への問い合わせ文を作成する"
  ],
  "nextQuestions": [
    "延長利用料はいくらですか？",
    "減免制度はありますか？",
    "夏休みだけ利用できますか？"
  ]
}
```

## 使い方

1. 通常どおり質問する
2. 回答後に表示されるチップをクリックする
3. 関連質問はそのまま再質問される
4. 「FAQとして保存」「未回答FAQとして保存」は管理機能に連携される

## ねらい

単発回答で終わらせず、利用者が次に確認すべきことを選べるようにすることで、FAQ検索を会話型AIアシスタントに近づけます。
