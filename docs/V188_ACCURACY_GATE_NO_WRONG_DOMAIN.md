# v188 Accuracy Gate No Wrong Domain

## 目的

「申請期限が過ぎてしまった」のように分野が明確な質問で、一般申請・勤務変更・就労要件など別分野のFAQを混ぜて回答しないようにする。

## 修正内容

- 明確な分野意図を hard strict intent として判定
- 該当分野FAQがない場合は、別分野FAQで無理に回答せず、正確性優先のフォールバック回答へ回す
- 関連FAQにも別分野候補が出にくくなるよう、strict filter のフォールバックを抑制

## 期待動作

質問: 申請期限が過ぎてしまった

期待回答: 受付可否・再申請・次回受付・担当窓口確認を案内する。子の看護、就労要件、通勤手当などは混ぜない。
