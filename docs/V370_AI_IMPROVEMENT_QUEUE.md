# V370 AI改善キュー / 回答評価 / 未回答自動登録

## 目的

v367〜v369で、Smart Assist は生成AI回答・根拠番号・DB行読み込み・DB条件抽出に対応した。
本バージョンでは、回答品質を継続改善するために、低信頼回答や根拠不足の回答を自動的に改善キューへ蓄積する。

## 追加内容

- `/smart-assist/improvement-queue` に `POST` を追加
- renderer API に `addSmartAssistImprovementQueue()` を追加
- AI回答後、以下の条件を満たす場合に改善キューへ自動登録
  - 根拠0件
  - `confidenceLevel === insufficient`
  - 信頼度43%未満
  - 生成AIが検索ベースへフォールバックし、根拠が少ない、または信頼度が低い
  - サーバー側から warning が返っている
- 回答カードに `📝 FAQ下書き` ボタンを追加
- 管理パネルに `改善キュー` タブを追加

## 安全設計

自動登録に失敗しても、回答表示は妨げない。保存失敗時は、利用者が手動で `👎 違う` を押すことで改善ログへ登録できる。

## 運用の流れ

1. 利用者が通常どおり質問する
2. 低信頼・根拠不足なら自動で改善キューへ登録
3. 管理パネルの `改善キュー` で確認
4. 必要に応じて再質問、FAQ下書き化、既存FAQの改善を行う

## 次の候補

- 改善キュー項目の status 更新
- キューから既存FAQへの紐付け
- キューから表記揺れ辞書・negativeTerms 候補の自動生成
- よく出る未回答質問のランキング化
