# v420: 未使用タグの安全な整理

## 追加内容

- タグ管理画面に「未使用タグの整理」を追加しました。
- ページ使用数が0で、共有別名辞書にだけ残っているタグを表示します。
- `辞書から外す` は共有タグ別名辞書の該当エントリだけを削除します。ページ本文、ページのタグ、履歴は変更しません。
- 実行前に確認ダイアログを表示します。
- 未使用タグは通常のタグ一覧・統合候補から除外し、作業対象が混ざらないようにしました。

## 仕様上の注意

このアプリでは、ページに使われていないタグは、別名辞書に登録されている場合だけ永続的に存在します。そのため、未使用タグの削除は「別名辞書から外す」操作として実装しています。
