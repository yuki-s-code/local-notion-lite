# v184 Domain Strict FAQ Answer Engine

## 目的

v183 までは、曖昧語・抽象語に対応するために汎用意図を広く拾っていたため、質問分野が違うFAQ文が回答に混ざることがありました。

例：

- 「有給はいつから使用できる？」に対して扶養手当・子の看護・通勤手当・就労要件が混ざる
- 「子どもが熱で休みたい」に対して就労要件が混ざる

v184 では、回答生成前に質問分野を強く判定し、違う分野のFAQを候補・回答文の両方から除外します。

## 主な改善

- 年次有給休暇 / 年休 / 有休 / 有給 の専用意図を追加
- 扶養手当、通勤手当、就労要件、子の看護を別ドメインとして分離
- サーバー側FAQ検索ランキングにドメイン分離スコアを追加
- チャット側回答生成にもドメイン分離フィルターを追加
- 回答文の最終生成直前に、別分野の箇条書きを削除

## 期待動作

質問：

```txt
有給はいつから使用できる？
```

期待回答：

```txt
年次有給休暇をいつから使用できるかは、採用日・勤務日数・勤務時間・継続勤務期間などの勤務条件により確認します。

・まず、勤務条件通知書や所属の運用で、付与日・付与日数・使用開始時期を確認します。
・パートタイムや会計年度任用職員の場合は、勤務日数・勤務時間・継続勤務期間により扱いが変わることがあります。
```

扶養手当、子の看護、通勤手当、就労要件の文章は混ざらない想定です。
