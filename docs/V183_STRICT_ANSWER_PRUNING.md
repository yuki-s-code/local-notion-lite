# v183 Strict Answer Pruning

## 目的

チャット回答で、質問意図と違うカテゴリの箇条書きが混ざる問題を修正します。

## 修正内容

- 子どもの発熱・子の看護・休暇系の質問では、回答生成の最終段階で就労要件・勤務時間・料金・減免・年休・忌引きなどの別カテゴリ文を除外します。
- `buildPersonalizedFaqAnswer()` で、子の看護系の質問を就労条件より先に判定します。
- サーバーFAQ検索結果から回答を構成する直前にも `cleanGeneratedAnswerLines()` を通し、最終表示でも不要文を落とします。

## 期待する挙動

「子供が熱で休みたい」と聞いた場合、回答は子の看護・所属への連絡・必要書類・事後提出に絞られ、就労要件や料金減免などは混ざりません。
