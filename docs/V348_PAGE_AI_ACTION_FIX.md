# V348 Page AI Action Fix

右下AIチャットの「このページを要約」「TODO抽出」などのページ操作で、通常のWorkspace検索候補が混入し、別FAQの内容を要約してしまう問題を修正しました。

## 変更点

- 「このページを要約」系の質問をページ要約Intentとして判定
- 「TODO抽出」「期限」「確認事項」系の質問をページTODO Intentとして判定
- ページ要約/TODOではSemantic Search候補を無理に混ぜず、現在ページ本文だけを優先
- 生成AIが失敗した場合も、検索候補ではなく現在ページ本文ベースのfallback回答を返す
- 生成AI失敗時の長いstderrを画面に直接出さず、短い警告に変更
- 関連情報を探すIntentの場合のみ、従来どおりSemantic Search候補を利用

## 期待される挙動

- 「このページを要約して」→ 現在開いているページの要約
- 「TODO抽出」→ 現在ページ本文からTODO/期限/確認事項候補を抽出
- 「このページに関連する情報を探して」→ Workspace Semantic候補を表示
- 生成AI失敗時 → ページ本文ベースの要約/抽出を表示し、長いllamaログは出さない
