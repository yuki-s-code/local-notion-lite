# v520: ページ・データベースを統一した作業スペースタブ

## 操作
- サイドバーのページをクリックすると、従来どおりページを開き、同じ作業スペースタブ列に登録します。
- サイドバーのデータベースをクリックすると、同じ作業スペースタブ列にDBタブとして登録・選択します。
- DBタブは主表示として閲覧専用で開きます。通常編集は「編集画面を開く」から明示的に移動します。
- ページ・DBともタブを右クリックして、ピン留め、閉じる、分割、比較に進めます。

## 安全性
- DBタブは読み取り専用であり、共有フォルダ上のDBロック、保存、履歴作成を開始しません。
- DBを編集する場合のみ、既存のDB編集画面と競合検出・保存処理を使用します。
