# v290 Smart Assist Compact Answer UI

## 目的

v289 では Smart Assist の回答に、確認事項・次アクション・関連質問・絞り込み・根拠・関連根拠候補をすべて展開表示していたため、FAQ回答としては情報量が多くなりすぎていた。

v290 では、実務画面で読みやすいように回答カードを整理した。

## 変更内容

### 1. 常時表示を最小化

常時表示する情報を次に絞った。

- 回答本文
- 信頼度
- 根拠FAQ
- 必要な場合のみ確認ポイント
- 参考候補は上位3件のみ

### 2. 重複表示を削減

以下は常時表示から外した。

- 関連して聞けること
- 絞り込み
- 次にできます

必要な場合は `質問候補・絞り込みを表示` の折りたたみ内にまとめる。

### 3. 参考候補の扱いを変更

`関連根拠候補` という表現は、回答根拠と誤解されやすいため、`参考候補` に変更した。

- 初期表示: 上位3件
- 折りたたみ展開: 最大8件
- スコア45未満は非表示
- 回答の根拠FAQと同一候補は可能な限り除外

### 4. 実務上の位置づけ

- `根拠FAQ`: 回答の主根拠
- `参考候補`: 追加確認・周辺確認用
- `確認ポイント`: 信頼度が低い、または質問が曖昧な場合のみ表示
