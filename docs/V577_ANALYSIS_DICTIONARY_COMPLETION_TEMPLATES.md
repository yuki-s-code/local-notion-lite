# v577 分析ノートブック：データ辞書・SQL補完・テンプレート

## 追加内容

- 分析用のデータ辞書 API
  - `pages`
  - `databases`
  - `database_rows`
  - `journals`
  - `tasks`
  - 各テーブルの列名、型、用途を返す
- 分析ノートブック左パネルを3タブ化
  - ノート
  - データ辞書
  - テンプレート
- SQLセルの補完
  - `Ctrl + Space` / `Command + Space` で候補表示
  - テーブル名・列名を候補に表示
  - 矢印キー、Enter、クリックで挿入
- よく使う分析テンプレート
  - データベース別の行数
  - Journal月次推移
  - 未完了タスク一覧
  - ページ更新の月次推移
- 条件セルのAPI受け渡し修正
  - 画面から送った `parameters` を Vault Service が DuckDB 実行へ正しく渡す

## 安全性

- SQLは引き続き `SELECT` / `WITH` の1文のみ
- DuckDBは端末側の `analysis.duckdb` のみを利用
- 分析ノートは `local.sqlite` に保存
- 共有フォルダの正本データは更新しない

## 既存ノート互換性

既存のSQLセル・メモセル・条件セル・グラフ設定の保存形式は維持する。
テンプレートは現在のノートを上書きせず、新しい未保存ノートとして開く。
