# v491: OCR実行ファイルの手動選択

## 追加内容

設定画面の「ローカルOCR」から、利用する `tesseract.exe` を手動で選択できるようにした。

## 探索順

1. ユーザーが設定画面で選択した実行ファイル
2. アプリに同梱した `resources/ocr/tesseract.exe`
3. OS の PATH にある `tesseract.exe`

選択した場所は各PCの Electron ユーザー設定へ保存され、共有フォルダやワークスペースデータには保存しない。選択直後から有効で、再起動は不要。

選択先が次回起動時に存在しない場合は、自動検出へフォールバックする。
