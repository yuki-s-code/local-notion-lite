# V357 AI Chat History

## 概要

右下AIチャットに、端末内保存のチャット履歴を追加しました。

## 追加内容

- AIチャット履歴ボタンを追加
- 新規チャット開始ボタンを追加
- 直近30件のAIチャットをlocalStorageへ自動保存
- 履歴から会話を再開
- ページ名、メッセージ件数、更新時刻を表示
- 履歴単位で削除
- 全履歴削除
- 回答モード、読込モード、回答長さ、対象範囲、固定/除外参照元も復元

## 保存場所

サーバー側DBには保存せず、まずは安全に端末のlocalStorageへ保存します。

```txt
local-notion-lite-ai-chat-history-v357
```

## 意図

v356までで、右下AIは参照元固定・回答保存・FAQ下書き化が可能になりました。
ただし、会話を閉じると過去の質問・回答を再利用しにくかったため、v357では「前回のAI相談を開き直して続ける」導線を追加しています。

## 注意

- 端末ごとの履歴です。
- 共有フォルダ全体で同期される履歴ではありません。
- 保存件数は最大30件です。
- 各会話は最大40メッセージまで保存します。
