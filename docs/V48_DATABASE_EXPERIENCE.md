# v48 Database Experience

## 目的

v47のデータベースUIをさらにNotion風にし、日常的な入力・確認・編集がしやすい画面に整理しました。

## 追加・改善

- データベース上部をコマンド型ヘッダーに刷新
- 行数、入力率、チェック済み、選択中行のインサイトカードを追加
- 行のローカル追加、複製、削除をテーブル内で操作可能に
- プロパティ編集パネルを追加
- プロパティ名の直接編集に対応
- プロパティのクイック追加に対応
- テーブル右側にRow previewを追加
- 選択行の詳細を右側で確認・編集可能
- Compact / Comfort 表示密度切替を追加
- 空状態をモダンなオンボーディング表示に変更
- 本文内のデータベース埋め込みカードも視認性を改善

## 設計方針

新しい重いUIライブラリは追加せず、現在のReact + BlockNote + Mantine構成を安定させたまま、CSSとコンポーネント分割で改善しています。

今後、本格的な列幅変更・列固定・複数行選択・仮想スクロールを入れる場合は、データベース部分だけTanStack Tableへ移行する方針が適しています。
