# V513 — 調整可能な分割・比較ペイン

## 変更内容

- 右側の「参照ページ」ペインを、左端のハンドルをドラッグして横幅変更できるようにした。
- 「ページを比較」の左右ペイン間にドラッグ可能な境界線を追加した。
- 参照ペイン幅と比較ペイン比率を `localStorage` に保存し、アプリ再起動後も端末ごとに復元する。
- 参照ペインの幅に合わせて編集領域の右余白も動的に調整する。
- 画面幅が900px以下の場合は、従来どおり縦レイアウトへ自動切替し、ドラッグ調整は無効にする。

## 制約

- 設定は各PCのUI設定であり、共有フォルダへ保存しない。
- 参照ペインの幅は330〜760pxの範囲とし、編集領域を最低限確保する。
- 比較表示は左右それぞれ30〜70%の範囲で調整できる。
