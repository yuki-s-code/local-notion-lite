# v204 Generic Term Subtopic Guard

## 目的

v203では「費用の確認をしたい」のような汎用語だけの質問で、減免FAQなどのサブトピックFAQが高信頼になるケースがありました。

v204では、汎用語だけではサブFAQを高信頼にしないためのガードを追加しました。

## 主な修正

- 「費用・料金・利用料」系の質問タイプ `fee_or_price` を追加
- 「減免・免除・非課税・兄弟・割引」系の質問タイプ `reduction_or_exemption` を追加
- 質問に減免語がない場合、減免FAQを強く減点
- 汎用費用質問では一般費用FAQを優先
- 汎用費用質問で減免FAQが1位になっても高信頼にしない上限処理を追加
- サンプルFAQの学童費用Intentを `afterschool.fee` に修正

## 期待される挙動

### 入力

```txt
費用の確認をしたい
```

### 改善後

- 減免FAQは高信頼になりにくい
- 一般費用FAQがあれば優先
- 一般費用FAQを特定できない場合は、中信頼または確認質問に落とす

### 入力

```txt
学童の減免制度はありますか
```

### 改善後

- 減免語があるため、減免FAQを正しく優先
