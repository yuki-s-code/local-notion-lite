# v194 Smart Assist Control Panel

## 目的
v193で追加した精度改善機能を、画面から迷わず操作できるようにしたバージョンです。
精度ロジック自体は大きく変えず、運用導線と確認性を改善しています。

## 追加・改善内容

- 右側上部に「Smart Assist 運用パネル」を追加
- FAQ検索インデックス再構築ボタンを運用パネルへ配置
- node-nlp再学習ボタンを追加
- 低信頼ログ確認ボタンを追加
- FAQ JSON取込を運用パネルからも実行可能に変更
- FAQカードに「テスト」ボタンを追加
- FAQ管理画面に Intent ID / Intent表示名 / テスト質問 の編集欄を追加
- 回答本文と確認質問を画面上で分離表示
- 高信頼判定を 85 点以上に調整
- 低信頼判定を 60 点未満に調整
- 高信頼時の関連候補表示を抑制し、1FAQ回答をより明確化

## 推奨運用

1. FAQ JSONを取り込む
2. FAQカードの「テスト」で想定質問が正しく当たるか確認する
3. Intent IDを `application.cancel` のように統一する
4. node-nlp再学習を実行する
5. 低信頼ログを確認し、足りないFAQやテスト質問を追加する
6. 再インデックスを実行する

## 信頼度基準

- 高: 85以上
- 中: 60〜84
- 低: 60未満

実務FAQでは誤回答を避けることを優先し、高信頼の閾値をv193より厳しめにしています。
