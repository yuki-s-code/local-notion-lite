# v153 AI Usability / Quality / PDF / Chat Memory Pack

## Summary
v153 strengthens the offline Local Generative Assist experience without adding Ollama or cloud AI.

## Added
- AI home cards for FAQ count, approved FAQ, PDF FAQ, and review-needed FAQ.
- Answer mode switcher: standard, short, detailed, steps, evidence-first, FAQ format.
- Dynamic recommended questions based on pinned questions, FAQ categories, PDF FAQ, and draft FAQ status.
- Pinned questions with local persistence.
- Optional local chat history persistence and clear history button.
- Improve answer modal: users can rewrite a bad answer and save it as an approved FAQ.
- FAQ quality tools: auto-categorize FAQ, review draft FAQ, detect and merge duplicate FAQ candidates.
- PDF knowledge view inside FAQ Builder.

## Design decision
The system remains fully offline and lightweight. It does not generate free-form LLM text; it improves answer assembly, evidence presentation, and FAQ growth workflows.
