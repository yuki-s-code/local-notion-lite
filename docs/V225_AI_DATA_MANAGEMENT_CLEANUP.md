# v225 AIデータ管理整理

## 目的
FAQ管理、質問統計、CSV入出力、言い換え辞書が分散して分かりにくかったため、AI関連データの入出力を「AIデータ」タブへ集約しました。

## 変更点
- 管理タブを「概要 / FAQ管理 / AIデータ / 質問統計 / 運用」に整理
- FAQ、評価セット、表記揺れ辞書、担当先、言い換え辞書、汎用ルール、回答フィードバックをAIデータ管理へ集約
- 各データにJSON取込/出力、CSV取込/出力のボタンを配置
- 各カードに「何を管理するか」「CSV列」を表示
- 運用操作は、再生成・正答率自動測定・低信頼ログ確認の3つに整理
- 旧NLP再学習、辞書編集の常時ボタン、FAQ JSON取込の重複導線を撤去

## 運用方針
- FAQを画面で編集したい場合は FAQ管理
- データを一括で差し替える場合は AIデータ
- 利用状況を見る場合は 質問統計
- 日常的なメンテナンスは 運用
