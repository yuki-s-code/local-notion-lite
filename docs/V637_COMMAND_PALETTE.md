# V637 Command palette enhancement

- Existing `Ctrl/Cmd + K` command palette is extended rather than duplicated.
- Recent pages, databases, and journals are surfaced from local-only recent history.
- A typed query can be handed directly to Workspace AI Search, covering body text, PDF, and OCR evidence when metadata titles do not match.
- The shortcut hint now uses `Ctrl K` on Windows and `⌘K` on macOS.
