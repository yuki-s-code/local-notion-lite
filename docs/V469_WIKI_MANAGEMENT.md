# V469 Wiki Management

Adds a Notion-style Wiki quality layer without a new database.

- Sidebar `📚 Wiki管理` opens a responsive management screen.
- Each page now supports Wiki status (draft / review / verified / archived), verification and review dates, owner, source, and successor page ID.
- Page properties remain stored through the existing page save, lock, history, and shared-folder flow.
- Semantic rebuilds include Wiki metadata. Workspace AI reranking boosts verified sources and strongly deprioritizes archived sources (archived items remain available for explicitly historical use).

Use the page's property panel to set detailed Wiki data. Use the Wiki management screen for triage and quick status changes.
