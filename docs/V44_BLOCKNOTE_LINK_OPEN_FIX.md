# v44 BlockNote標準リンクUIからのページプレビュー修正

## 変更点

- BlockNote標準リンクUIの「ページを開く」を押したとき、右サイドバーのページプレビューが開くように修正しました。
- 独自ツールチップは復活させず、リンク編集/ページを開くはBlockNote標準UIに寄せています。
- `@` 候補では、インラインリンクとカード型リンクを明示的に選べるように表示文言を整理しました。
- `local-page://...` と `#local-page=...` の両方を引き続き読み取り互換として扱います。

## 方針

BlockNote標準のリンクツールチップには「リンク編集」「ページを開く」を任せます。
表示形式の選択は、リンク作成時の `@` 候補で「インライン」「カード」を選ぶ設計にしています。
