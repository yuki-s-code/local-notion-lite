# V516 — Horizontal database tab rail

The database navigator was redesigned as a single-line, browser-like horizontal tab rail.

- Long titles truncate on one line and retain their full name in the native tooltip.
- The tab rail is horizontally scrollable and includes previous/next controls.
- Database type and row/view identity remain available through icons and the tooltip instead of a second visible text line.
- View controls are separate compact chips underneath the main rail.
- The existing single editable DatabaseTable remains the only editor, so shared-folder locking and autosave behavior are unchanged.
