# v110 Backup / Restore Center

## 追加内容

- バックアップ・復元センター画面を追加
- ページ履歴、削除済みページ、削除済みDB、削除済みJournalを一覧化
- バックアップサイズ、作成日時、種類を表示
- 復元ボタンを追加
- 共有フォルダ管理からバックアップ・復元センターへ移動可能

## 方針

共有フォルダ運用では、便利機能よりも「戻せる安心感」が重要です。
そのため、既存の backups / conflicts / trash / attachments を一箇所で確認できる管理画面として実装しています。

## 注意

ページ履歴の復元は既存のページ履歴復元機能を利用します。
削除済みデータの復元は backups 配下に退避されたデータを共有フォルダへ戻します。
