# v47 Modern Database UI

## 目的
データベース画面と本文中のデータベース埋め込みを、Notion風の視認性・操作感に近づけるためのUI改善版です。

## 追加・改善

- データベースヘッダーをカード型のヒーローUIへ変更
- 行数・入力率・チェック済み件数をサマリー表示
- ビューをタブ型に変更
- DB内検索ボックスを追加
- Filter / Sort を折りたたみ操作へ変更
- プロパティ一覧をピル表示
- テーブルヘッダーを固定風のモダン表示へ調整
- チェックボックス、select、空セルの見た目を改善
- 本文内のDB埋め込みも同じモダンテーブル表示を使用

## 方針
新しいUIライブラリはこの段階では追加していません。Electron + BlockNote + Mantine の依存関係を安定させることを優先し、CSSとReact構造でモダン化しています。

将来的にさらに進める場合は、Radix UI / Floating UI / TanStack Table の導入が候補です。
