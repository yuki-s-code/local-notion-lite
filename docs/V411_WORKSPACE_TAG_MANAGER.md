# v411 ワークスペースタグ管理

## 目的

タグ候補・別名機能を運用しやすくするため、通常ページのプロパティ内からワークスペース全体のタグを確認し、表記変更・統合を行えるようにする。

## 操作

- `ワークスペースのタグを管理` を開く。
- 使用ページ数と端末ローカルに保存されている別名を確認する。
- 対象タグを選び、新しいタグ名または統合先を入力する。
- `名前を変更` または `既存タグへ統合` を選ぶ。

## 安全性

- 実行前に対象ページ数を確認する。
- 更新対象は、選択タグを持つページの `properties.tags` のみ。
- 本文、BlockNoteデータ、コメント、履歴は変更しない。
- ページごとに `baseUpdatedAt` を付けて保存するため、別PCの更新があれば競合として当該ページだけを止める。
- 成功分だけ別名も統合先へ移す。
- タグの自動付与は行わない。

## 注意

別名は現在も端末ローカルの localStorage に保存される。共有タグ辞書へ昇格させるのは、利用者間で合意した後の別機能とする。
