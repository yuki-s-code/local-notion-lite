# v179 Local FAQ Generated Answer + Clean UI

## 目的
FAQ検索結果をそのまま表示するのではなく、ローカルFAQを根拠にした「回答生成」に寄せる。

## 方針
- 外部AIや重いLLMは使わない
- kuromoji.js + Fuse.js + 同義語辞書 + 意図判定の検索結果を根拠にする
- 回答本文には回答だけを表示する
- 根拠FAQ・関連FAQ・一致理由などの詳細は、既存のクリック可能な関連ソースボタンへ退避する

## 変更内容
- チャット本文から以下を削除
  - 信頼度見出し
  - 質問の再掲
  - 判定した意図
  - 根拠FAQの長文表示
  - 一致理由・一致語
  - 検索方式
  - 注意書き
- 回答本文は生成済み回答だけに整理
- 関連FAQは `sources` に渡し、UI側のクリック可能なボタンとして表示

## 効果
- チャットが読みやすくなる
- 回答に集中できる
- 詳細確認したい場合だけ関連FAQをクリックできる
