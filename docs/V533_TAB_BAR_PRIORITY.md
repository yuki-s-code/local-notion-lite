# v533 — タブ列を作業領域の最上段へ

## 目的
ページ・データベース・DB行の作業対象を切り替えるタブ列を、コメント・共有・ゴミ箱などの対象別操作より上に配置する。

## 変更
- `WorkspaceWorkbench` がページ固有の操作バーを `toolbar` スロットとして受け取り、タブ列の直下に配置する。
- ページタブ選択時は従来どおり、コメント・お気に入り・複製・ゴミ箱・Private/Shared・BlockNote・付箋を表示する。
- DBタブ選択時は、元ページを誤操作しないようページ固有ボタンを隠し、DBの固有操作（Private/Shared・分割・比較・削除）をDB見出し側に集約する。
- DBフォーカス時にページ本文を隠す既存のCSSと衝突しないよう、操作バーを作業スペース内部に配置する。
- タブ列は広い画面では上端にsticky表示、狭い画面では通常配置に戻す。

## 検証
`tsc --noEmit` は添付ZIPに `node_modules` がないため Electron/Node の型定義不足で停止する。変更したTSXの構造確認は実施済み。
