# v228 Unified FAQ Studio

## 目的

「FAQ管理」と「FAQを育てる」が似た役割で分かりにくかったため、FAQ関連の作業を「FAQ管理」タブへ統合した。

## 変更内容

- FAQを育てる専用モーダルを廃止
- FAQ管理タブに以下を統合
  - FAQ一覧
  - 改善キュー件数
  - 品質スコア
  - 出典なしの警告
  - 評価セット自動生成
  - 正答率測定
  - 検索再生成
  - 質問統計への導線
- JSON/CSV取込・出力はAIデータタブに集約
- FAQ編集は専用モーダルのまま維持
- 一覧はカード型で、質問・回答要約・品質・出典・操作を見やすくした

## 画面の役割

- FAQ管理: FAQを追加・編集・改善・テストする日常作業画面
- AIデータ: JSON/CSVでデータを取込・出力する管理画面
- 質問統計: 利用傾向を見る画面
- 運用: 再生成や低信頼ログ確認などの保守操作
