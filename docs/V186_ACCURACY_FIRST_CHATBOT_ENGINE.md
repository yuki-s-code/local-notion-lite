# v186 Accuracy First Chatbot Engine

正確性を最優先するため、FAQチャットの回答生成を次の順序に整理しました。

1. 質問からドメインを判定
2. 判定ドメインに合わないFAQを回答生成前に除外
3. 一致する根拠FAQだけで回答文を生成
4. 回答表示直前に別ドメインの文を最終除外
5. 根拠が弱い場合は断定せず、不足回答として関連候補を提示

## 目的

「有給はいつから？」に扶養手当・通勤手当・子の看護などが混ざる問題を避けるため、回答生成前のDomain Gateを強化しています。

## 追加した主な考え方

- 年次有給休暇
- 子の看護・休暇
- 期限超過・申請遅れ
- 変更・修正
- 就労要件
- 扶養手当
- 通勤手当
- 料金・減免
- 必要書類
- 手続き・申請

それぞれに positive / negative keywords を持たせ、異なる分野のFAQや文を回答に混ぜないようにしています。

## 正確性の方針

根拠が弱い場合は無理に高信頼回答を作らず、信頼度を下げて「確認してください」という回答にします。
