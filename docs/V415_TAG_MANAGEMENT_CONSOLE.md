# v415 Tag Management Console

The workspace tag screen is redesigned as a dedicated management console.

- Summary cards show tag inventory, tagged pages, assignment count, and local aliases.
- Search supports both canonical tag names and aliases.
- Tags can be sorted by usage or name.
- The selected tag displays aliases, rename/merge controls, and recently updated tagged pages.
- Rename and merge continue to modify tags only; body, comments, and history are not changed.
- Tag-only changes remain excluded from page history checkpoints.
